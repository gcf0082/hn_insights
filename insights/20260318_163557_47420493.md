# 洞察总结：Agent-Based Task Execution

**来源**: https://amaiya.github.io/onprem/examples_agent.html  
**时间**: 2026-03-18

## 核心内容

### 1. AgentExecutor 概述
- OnPrem.LLM 的 `AgentExecutor` 用于创建自主AI代理执行复杂任务
- 支持任何 LiteLLM 支持的模型（支持 tool-calling）
- 云端模型：OpenAI GPT-5.2-codex、Claude Sonnet 4.5、Gemini 1.5 Pro
- 本地模型：Ollama、vLLM、llama.cpp

### 2. 默认工具集（9个）
| 工具 | 功能 |
|------|------|
| read_file | 读取文件完整内容 |
| read_lines | 读取文件特定行范围 |
| edit_file | 通过查找/替换编辑文件 |
| write_file | 写入完整文件内容 |
| grep | 在文件中搜索模式 |
| find | 通过glob模式查找文件 |
| run_shell | 执行shell命令 |
| web_search | 网络搜索 |
| web_fetch | 抓取URL内容 |

### 3. 使用示例

#### 示例1：基础计算器（沙箱执行）
- 在隔离Docker/Podman容器中运行代理
- 创建 calculator.py 和 test_calculator.py
- 21个测试全部通过
- 成本：$0.0547，5次LLM调用

#### 示例2：网络研究代理
- 仅启用 web_search、web_fetch、write_file 工具
- 研究2026量子计算最新进展
- 生成38个参考文献的完整报告
- 成本：$0.4316

#### 示例3：本地模型
- 使用 glm-4.7-flash（Ollama）
- 通过网页抓取提取信息
- 建议上下文窗口至少8192token

#### 示例4：沙箱执行
- 使用 Docker/Podman 容器隔离
- 适合运行不受信任代码
- 启动开销约5-10秒

#### 示例5：自定义工具
- 可添加任意Python函数作为工具
- 示例：股票分析工具（yfinance）
- 自定义工具：get_current_stock_price、analyze_volatility、calculate_return_percentage

### 4. 安全特性
- `disable_shell=True`：移除shell访问
- `sandbox=True`：在临时容器中运行
- 文件访问限制在工作目录内
- 网络配置：Linux用network='host'，macOS/Windows用host.docker.internal

### 5. 关键参数
```python
AgentExecutor(
    model='anthropic/claude-sonnet-4-5',
    enabled_tools=['web_search', 'web_fetch'],  # 工具白名单
    disable_shell=True,  # 安全模式
    sandbox=True,  # 沙箱执行
    max_iterations=10,  # 最大迭代次数
    working_dir='./project'  # 工作目录
)
```

## 技术要点
1. 基于 PatchPal 编码代理实现
2. 支持云端和本地模型无缝切换
3. 灵活的工具配置机制
4. 企业级安全隔离选项
5. 自动完成检测（COMPLETE promise）
