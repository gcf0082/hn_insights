# 洞察总结：Agent-Based Task Execution

## 核心概述

OnPrem.LLM 的 `AgentExecutor` 是一个基于 PatchPal 的自主代理执行框架，支持云端和本地模型（通过 LiteLLM），可调用工具执行复杂任务。

## 关键功能

### 1. 内置9种默认工具
- 文件操作：`read_file`、`read_lines`、`edit_file`、`write_file`
- 搜索：`grep`、`find`（glob模式）
- 执行：`run_shell`
- 网络：`web_search`、`web_fetch`

### 2. 模型兼容性
- **云端**：OpenAI GPT系列、Anthropic Claude、Google Gemini
- **本地**：Ollama (ollama/llama3.1)、vLLM、llama.cpp

### 3. 安全特性
- **工作目录限制**：代理只能读写指定目录
- **沙箱执行**：`sandbox=True` 在Docker/Podman容器中运行，约5-10秒启动开销
- **工具控制**：`disable_shell=True` 可禁用shell访问

### 4. 自定义工具
- 通过Python函数定义自定义工具
- 自动保存到 `.patchpal/tools/` 目录

## 示例场景

| 示例 | 场景 | 工具配置 | 关键结果 |
|------|------|----------|----------|
| 计算器 | 创建Python模块+测试 | sandbox=True | 21个pytest测试全部通过，$0.0547 |
| 网络研究 | 量子计算2026报告 | web_search/web_fetch/write_file | 生成38条参考文献的综合报告 |
| 本地模型 | 网页信息提取 | Ollama glm-4.7-flash | 成功提取Ph.D.信息 |
| 沙箱数据 | 销售数据分析 | sandbox=True | 生成图表+分析报告 |
| 自定义工具 | 股票分析(AAPL/MSFT) | yfinance自定义工具 | 实时价格+波动性分析 |

## 成本效率
- Claude Sonnet 4-5：约 $0.005-0.05/任务
- Token消耗：单次任务 6,000-112,000 tokens

## 本地模型配置要点
- **上下文窗口**：建议至少 8192 tokens，推理模型可能需要 32K-64K
- **网络配置**：
  - Linux/WSL2：使用 `network='host'`
  - macOS/Windows：设置 `OLLAMA_API_BASE` 环境变量

## 技术栈依赖
- `patchpal`：代理执行核心
- `yfinance`：金融工具示例
- Docker/Podman：沙箱执行
