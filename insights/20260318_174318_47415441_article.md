# GPT-5.4 mini 与 nano 洞察报告

**来源**: OpenAI 官方公告  
**发布日期**: 2026年3月17日  
**报告日期**: 2026年3月18日

---

## 核心摘要

OpenAI 于 2026 年 3 月 17 日发布了 GPT-5.4 mini 和 GPT-5.4 nano，这是其迄今为止最强大的小型模型系列。这两款模型将 GPT-5.4 的核心优势带入更高效、更快速的版本，专为高频、对延迟敏感的工作负载设计，显著缩小了与旗舰模型的性能差距，同时大幅降低成本。

---

## 主要发布内容

### GPT-5.4 mini

- **性能提升**: 在编码、推理、多模态理解和工具调用方面相比 GPT-5 mini 显著提升
- **速度优势**: 运行速度提升 **超过 2 倍**
- **基准测试表现**:
  - SWE-Bench Pro: 54.4%（GPT-5 mini 为 45.7%，GPT-5.4 满血版为 57.7%）
  - OSWorld-Verified: 72.1%（满血版为 76.8%）
- **定位**: 接近旗舰模型的性能，适合复杂专业工作

### GPT-5.4 nano

- **定位**: 最小、最便宜、最快速的版本
- **适用场景**: 分类、数据提取、排序、简单编码任务
- **性能提升**: 相比 GPT-5 nano 有显著升级
- **上下文窗口**: 400,000 tokens

## API 定价

| 模型 | 输入价格 (每百万 tokens) | 输出价格 (每百万 tokens) |
|------|-------------------------|-------------------------|
| GPT-5.4 mini | $1.00 | $5.00 |
| GPT-5.4 nano | $0.20 | $1.25 |

**对比**: GPT-5.4 满血版输入为 $2.50，输出为 $15.00

---

## 速度性能

根据实际 API 测试：
- **GPT-5.4 mini**: 平均 180-190 tokens/秒
- **GPT-5.4 nano**: 约 200 tokens/秒
- 相比之下，GPT-5 mini 约 55-60 tokens/秒（普通模式），115-120 tokens/秒（优先模式）

---

## 技术特点

1. **多模态能力**: 支持文本和图像输入
2. **工具调用**: 内置可靠的工具调用能力
3. **低延迟**: 针对高频工专门优化，如代码助手、截图解析、实时图像推理
4. **子代理架构**: 适合与大型规划模型配合使用，构建混合代理系统

---

## 市场意义

1. **推动 Agent 架构**: 小型高效模型使得构建多模型代理架构成为可能，大模型负责规划，小模型负责执行
2. **成本优化**: 约 1/3 的价格获得接近满血版的性能
3. **应用场景扩展**: 让更多需要低延迟和低成本的生产环境能够使用 AI

---

## 可用性

- **GPT-5.4 mini**: 通过 API 和 ChatGPT 提供
- **GPT-5.4 nano**: 仅通过 API 向开发者开放
