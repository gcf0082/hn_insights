# Sashiko: Google 推出的 Linux 内核 AI 代码审查工具洞察

## 项目概述
- **名称**: Sashiko
- **开发者**: Google 工程师（由 Roman Gushchin 领导的 Google Linux 内核团队）
- **性质**: 代理式(Agentic) AI 代码审查系统
- **开源**: 是，托管于 GitHub (sashiko-dev/sashiko)
- **项目主页**: https://sashiko.dev/

## 核心亮点

### 性能表现
- **Bug 发现率**: 53%（基于1000个最近的上游 Linux 内核 "Fixes:" 标签问题）
- **关键洞察**: 虽然 53% 看似不高，但 100% 这些问题都被人类审查者遗漏了
- **测试模型**: Google Gemini 3.1 Pro
- **兼容性**: 设计用于 Gemini Pro 3.1，但应该也能与 Claude 等其他 LLM 配合使用

### 资金与基础设施
- **资助方**: Google 负责 Sashiko 的 token 预算和基础设施
- **项目托管**: 正在迁移至 Linux Foundation

## 关键价值

1. **自动化持续审查**: 目前已覆盖所有提交到 Linux 内核邮件列表的代码
2. **开源协作**: 代码公开，允许社区参与和改进
3. **AI + 人类协作**: AI 发现人类审查者遗漏的问题，补充而非替代人类审查

## 技术特点
- 代理式(Agentic)系统，可自主执行复杂的代码审查任务
- 针对 Linux 内核代码库进行专门优化
- 支持多种大语言模型后端
