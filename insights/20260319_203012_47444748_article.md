## 发现的问题

### P1 问题：User-Agent 头意外移除

在 `llm.ts` 的头重构中，原本发送给所有非 opencode、非 anthropic provider 的 `User-Agent: opencode/${VERSION}` 头被静默移除。这可能是一个非故意的副作用，影响 OpenAI、Google、Azure、OpenRouter 等提供商。

### P2 问题：文档不一致

`providers.mdx` 第 2 步的代码块正确更新为仅显示 `Manually enter API Key`，但其上方的描述仍提到"Claude Pro/Max 选项"，与已更新的 UI 不一致。
