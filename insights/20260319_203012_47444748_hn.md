## 事件概述

Anthropic 向 OpenCode 发出法律请求，要求其删除允许用户使用 Claude Code 订阅 Token 的功能。OpenCode 是一个类似于 Claude Code 的 AI 编程 CLI 工具，支持多种 LLM 提供商。

## 核心争议

1. **订阅Token使用限制**: Anthropic 的订阅计划（如 Pro/Max）仅限第一方客户端使用，用户不应将订阅 Token 用于第三方工具
2. **API 差异**: Anthropic 区分 Claude Code 订阅 Token 和标准 API Key，前者仅限 Claude Code 使用，后者可自由用于任何客户端
3. **删除的代码**: OpenCode 移除了 `opencode-anthropic-auth` 认证插件及相关提示文本

## 社区观点

### 支持 Anthropic
- 订阅计划是补贴性质（比 API 便宜约 90%），用户应在规定范围内使用
- 企业有权决定自己的服务条款和商业模式
- 要求第三方工具停止帮助用户违反 ToS 是合理的

### 批评 Anthropic
- 这是反竞争行为，试图锁定用户使用 Claude Code
- 过于激进，与开源社区关系紧张
- 担忧 Anthropic 收购 Bun 后的潜在影响

## 技术细节

- Anthropic 通过 Claude Code 实现了超过 95% 的缓存命中率
- 订阅计划的 Token 配额约为 API 同价格的 8-12 倍
- 这是 Anthropic 近期第二次对第三方客户端采取行动（之前是 OpenClaw）
