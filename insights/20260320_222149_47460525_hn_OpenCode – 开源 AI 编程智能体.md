## 项目简介

OpenCode 是一个开源的 AI 编程代理，旨在为开发者提供一个可定制、无供应商锁定的编程辅助工具。

## 用户反馈要点

### 1. 多模型支持
- 可与 Claude、GPT、Qwen、Llama.cpp、Gemini 等多种模型配合使用
- 支持本地模型运行（如 Qwen 3.5）
- 可灵活选择不同模型用于不同任务

### 2. 高度可定制性
- 支持 skills、FastAPI server 集成
- 强大的 LSP 集成
- 用户可构建插件系统（如 OpenClaw 替代品）
- 代码组织结构优秀，值得学习

### 3. 成本优势
- 提供免费层，可用于日常任务
- 比 API 调用更经济实惠
- 配合 Kimi、Fireworks AI 等提供商速度快

### 4. 替代方案
- 部分用户从 Aider 转向 OpenCode
- 可作为 Claude Code 的替代方案

## 技术讨论

### API vs 订阅
- 订阅制：需每日刷新 token 1-2 次，但可自由使用配额
- API：更昂贵但使用更灵活

### 注意事项
- 部分用户提及 telemetry 默认开启
- 不支持禁用流式传输（部分推理提供商不可用）
