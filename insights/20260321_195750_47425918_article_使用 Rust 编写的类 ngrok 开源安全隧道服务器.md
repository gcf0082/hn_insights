## 核心功能

- **HTTP/TCP 代理**：通过公共服务器暴露本地服务
- **TLS 终止**：安全的 WebSocket 连接
- **实时仪表板**：监控隧道状态
- **Prometheus 指标**：性能监控
- **审计日志**：记录认证和隧道活动
- **MCP 服务器**：AI 代理集成（支持 Claude、GPT-4o 等）
- **自托管或托管服务**：可自行部署或使用 `edge.rustunnel.com`

## 技术架构

```
Internet ──── :80 ─────▶│  HTTP edge (301 → HTTPS)
Internet ──── :443 ────▶│  HTTPS edge ──▶ yamux stream ──▶ client
Client ───── :4040 ────▶│  Control-plane WebSocket (TLS)
Browser ──── :8443 ────▶│  Dashboard UI + REST API
Prometheus ─ :9090 ────▶│  Metrics endpoint
Internet ── :20000+ ───▶│  TCP tunnel ports
```

## 技术要求

### 构建要求
- Rust toolchain 1.76+
- pkg-config
- libssl-dev
- Node.js + npm (仅需重建仪表板 UI)

### 生产环境要求
- Linux (Ubuntu 22.04+)
- TLS 证书 + 私钥
- 公共 IP / DNS（HTTP 隧道需要通配符 DNS）

## 部署方式

1. **本地开发**：Docker Compose
2. **生产环境**：systemd 或 Docker
3. **托管服务**：edge.rustunnel.com

## 客户端安装

- **Homebrew** (macOS/Linux): `brew tap joaoh82/rustunnel && brew install rustunnel`
- **预编译二进制**：从 GitHub Releases 下载
- **源码构建**：Rust 1.76+ 编译

## 监控

- Prometheus 指标端点：`:9090/metrics`
- Grafana 仪表板支持
- 关键指标：
  - `rustunnel_active_sessions`
  - `rustunnel_active_tunnels_http`
  - `rustunnel_active_tunnels_tcp`

## REST API

仪表板端口暴露 REST API，用于编程访问隧道、令牌、捕获的请求和隧道历史。

## AI 代理集成

提供 MCP 服务器实现，允许 AI 代理（如 Claude）无手动干预地打开和管理隧道。
