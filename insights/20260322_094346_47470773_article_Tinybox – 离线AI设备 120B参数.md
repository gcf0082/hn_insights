## Basic Information

### About tinygrad
- **Type**: Simple and powerful neural network framework
- **Status**: Growing fastest neural network framework
- **Key Features**:
  - Extremely simple design
  - Breaks down complex networks into just 3 OpTypes
  - Supports full forward and backward passes with autodiff
  - Similar API to PyTorch but simpler and more refined
  - Used in openpilot for driving model on Snapdragon 845 GPU

### 3 OpTypes Architecture
1. **ElementwiseOps**: UnaryOps, BinaryOps, TernaryOps (SQRT, LOG2, ADD, MUL, WHERE, etc.)
2. **ReduceOps**: Return smaller tensor (SUM, MAX)
3. **MovementOps**: Virtual ops that move data around (RESHAPE, PERMUTE, EXPAND)
