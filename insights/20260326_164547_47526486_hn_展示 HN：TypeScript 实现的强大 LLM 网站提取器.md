## 项目概述

**Lightfeed Extractor** 是一个 TypeScript 库，专门用于从原始 HTML 中提取结构化数据。该项目解决了传统网页爬虫面临的常见痛点，包括 CSS 选择器失效、HTML 噪音干扰、JSON 解析错误等问题。

## 核心功能

1. **HTML 清理与转换**: 将 HTML 转换为适合 LLM 处理的 Markdown 格式，自动去除导航栏、页脚等噪音内容，支持可选的图片提取和 URL 清理

2. **多 LLM 支持**: 兼容任何 LangChain 支持的 LLM（OpenAI、Gemini、Claude、Ollama 等）

3. **类型安全提取**: 使用 Zod schema 进行类型安全的结构化数据提取，带有实时验证功能

4. **错误恢复机制**: 从格式错误的 LLM 输出中恢复部分数据，即使部分解析失败也能获取有效结果

5. **内置浏览器自动化**: 通过 Playwright 支持本地、无服务器或远程浏览器操作，并包含反机器人保护

6. **AI 驱动导航**: 可配合 Lightfeed 浏览器代理实现 AI 驱动的页面导航

## 技术亮点

- 采用 Apache 2.0 开源许可证
- 支持自定义清理规则和输出格式
- 可处理相对 URL、Markdown 转义链接、追踪参数等细节问题

## 适用场景

该库特别适合需要大规模网页数据提取的项目，尤其是需要处理多样化网页结构且希望降低维护成本的团队。
