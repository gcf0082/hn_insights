# 洞察报告：Anthropic 调整 Claude 使用限制

## 基本信息

- **洞察链接**: https://twitter.com/trq212/status/2037254607001559305
- **发布来源**: Thariq Shihipar (Anthropic 技术团队成员)
- **发布日期**: 2026年3月26日
- **主题**: Claude 使用限制调整

## 洞察内容

Anthropic 于2026年3月26日宣布调整 Claude 用户的会话使用限制，以应对不断增长的需求。

### 主要变更

1. **限时会话调整**: 在高峰时段（太平洋时间 05:00-11:00，即 GMT 13:00-19:00），5小时的会话限制可能会在不到5小时内用完
2. **周限额保持不变**: 用户的每周使用限额没有变化
3. **影响范围**: 约7%的用户将遇到此前不会遇到的会话限制，特别是 Pro 订阅用户

### 用户建议

- 如果运行 Token 密集型的后台任务，建议将其移至非高峰时段执行，以延长会话限制的使用时间
- 总体每周限额保持不变，只是如何在一周内分配有所变化

### 背景

Anthropic 通过不公开的限额来管理 Claude 的使用，用户无法提前规划 Token 使用量。订阅用户（Free、Pro、Max）只能在这些未公开的限制下使用 Claude。
