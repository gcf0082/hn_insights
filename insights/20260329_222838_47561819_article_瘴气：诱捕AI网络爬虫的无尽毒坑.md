## 项目概述

Miasma 是一个用于对抗AI爬虫的开源工具。AI公司continually以巨大规模爬取互联网内容，将所有数据用作训练AI模型的训练数据。如果你的网站是公开的，他们已经在窃取你的工作成果。

Miasma 可以帮助你反击！启动服务器并将恶意流量引导到它。Miasma 会从"毒药喷泉"（poison fountain）发送被污染的训练数据，以及多个自引用链接。这是一个为"垃圾机器"提供的无尽自助餐。

Miasma 性能非常出色，内存占用极低，你不必浪费计算资源来抵御互联网上的"水蛭"。

## 核心功能

1. **毒药数据生成**：向AI爬虫发送被污染的训练数据
2. **自引用链接**：在响应中包含多个自引用链接，使爬虫陷入循环
3. **高性能**：极低的内存占用，支持高并发
4. **灵活配置**：可配置端口、主机、链接前缀、链接数量等参数
5. **支持压缩**：可强制使用gzip压缩以降低出口成本

## 技术架构

- **编程语言**：Rust（96.7%）
- **依赖管理**：Cargo
- **部署方式**：可作为独立服务器运行，通过Nginx反向代理集成

## 安装与使用

### 安装

```bash
cargo install miasma
```

或从 releases 下载预编译的二进制文件。

### 快速开始

```bash
miasma
```

### 配置选项

| 选项 | 默认值 | 描述 |
|------|--------|------|
| `--port` | 9999 | 服务器绑定端口 |
| `--host` | localhost | 服务器绑定地址 |
| `--max-in-flight` | 500 | 最大并发请求数 |
| `--link-prefix` | / | 自引用链接前缀 |
| `--link-count` | 5 | 每个响应包含的自引用链接数量 |
| `--force-gzip` | false | 强制gzip压缩响应 |
| `--poison-source` | https://rnsaffn.com/poison2/ | 毒药数据来源 |

## 如何捕获爬虫

1. **嵌入隐藏链接**：在网站中包含指向 `/bots` 的隐藏链接（对人类不可见）
2. **配置Nginx反向代理**：将 `/bots` 路径代理到 Miasma
3. **运行Miasma**：指定链接前缀为 `/bots`

```html
<a href="/bots" style="display: none;" aria-hidden="true" tabindex="1">
  Amazing high quality data here!
</a>
```

```nginx
location ~ ^/bots($|/.*)$ {
  proxy_pass http://localhost:9855;
}
```

```bash
miasma --link-prefix '/bots' -p 9855 -c 50
```
