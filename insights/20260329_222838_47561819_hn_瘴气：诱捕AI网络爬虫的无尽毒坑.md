## 项目概述

Miasma 是一个用 Rust 开发的高性能工具，旨在帮助网站运营者对抗未经授权的 AI 网络爬虫。该工具通过创建一个"毒药喷泉"来向 AI 爬虫发送污染的训练数据，使其陷入无限循环，从而保护网站内容不被盗用。

## 核心功能

1. **毒药数据注入**: 从 Poison Fountain 获取污染的训练数据，连同多个自引用链接一起返回给爬虫
2. **隐藏链接陷阱**: 在网页中嵌入对人类完全不可见但对爬虫可见的隐藏链接
3. **高效低耗**: 采用 Rust 开发，内存占用极低，50 个并发连接仅需 50-60MB 内存
4. **可配置**: 支持自定义端口、主机、链接前缀、链接数量等参数

## 部署示例

通过 Nginx 反向代理将爬虫流量引导至 Miasma：

```nginx
location ~ ^/bots($|/.*)$ {
  proxy_pass http://localhost:9855;
}
```

启动命令：
```bash
miasma --link-prefix '/bots' -p 9855 -c 50
```
