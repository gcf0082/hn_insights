# Claude Code Unpacked：可视化指南

## 基本信息

- **来源网站**：ccunpacked.dev
- **作者**：autocracy101
- **发布时间**：1小时前

## 内容摘要

这是一个可视化指南网站，展示了 Claude Code 内部的架构和工作流程。该网站基于泄露的 Claude Code 源码（约50万行代码）进行可视化分析。

### 核心内容

1. **Agent Loop（代理循环）**：从用户输入到响应的完整流程，共11个步骤
2. **架构探索器**：564个工具文件、389个组件、189个命令等
3. **工具系统**：涵盖文件操作、执行、搜索、代理任务、规划、MCP等多个类别
4. **命令目录**：包含72个斜杠命令，分为设置、日常工作、代码审查、调试等类别
5. **隐藏功能**：未发布的功能，包括 Buddy（虚拟宠物）、Kairos（持久模式）、UltraPlan、Coordinator Mode、Bridge、Daemon Mode 等

## 社区反馈要点

- **代码规模**：约50万行代码，由 AI 辅助编写
- **代码质量争议**：有评论者认为代码"vibe-coded"（凭感觉编码），存在技术债务
- **未解决问题**：会话间的记忆整合（memory consolidation）仍是未解决的难题
- **希望功能**：用户期待跨会话引用和 /Buddy 功能能移植到 OpenCode
- **动画效果**：有评论认为动画过于花哨，实用信息有限
