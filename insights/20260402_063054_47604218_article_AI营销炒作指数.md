# AI营销废话指数（AI Marketing BS Index）

**洞察链接**: https://bastian.rieck.me/blog/2026/bs/  
**发布日期**: 2026年4月1日  
**作者**: Bastian Rieck  
**标签**: research, musings  

---

## 摘要

本文作者提出了一个用于评估AI营销文案是否包含"废话"的评分系统——"AI营销废话指数"。该评分标准借鉴了John Baez的"民科指数"（Crackpot Index），旨在量化评估那些过度使用营销术语、滥用专业术语的AI产品宣传文案。

---

## 背景

作者指出，当前AI领域的营销文案存在严重问题：每一家真正想推广好产品的公司，往往伴随着三家"把氛围感拉满"的公司，它们滥用各种专业术语。因此，作者创建了这个评分系统来分类和评估营销材料。

---

## 评分标准

| 分数 | 评分项 |
|------|--------|
| -5分 | 概念从-5分开始（给所有人一个先发优势） |
| +10分 | "发明"某物但未提供任何引用、论文或规范 |
| +10分 | 每出现一次来自数学、物理或生命科学但不应出现的术语 |
| +20分 | 使用典型的"城堡-护城河"（motte-and-bailey）逻辑或 hedging，如"这不是X，是Y" |
| +20分 | 以伪深刻的无意义语句结束段落 |
| +20分 | 声称产品做的是"自然"或"宇宙"所做的事 |
| +20分 | 在不必要的情况下提及"涌现特性" |
| +20分 | 每提及一次常春藤名校 |
| +30分 | 在"技术"描述中完全没有可证伪的主张或预测 |
| +40分 | 每提及一个无法验证的"研究合作" |

---

## 结论

作者表示，随着时间推移，这个评分标准可能会继续扩展。同时，作者希望读者的收件箱里能少一些空洞的长邮件。
