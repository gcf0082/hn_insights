# Claude 额外使用积分赠送活动

## 基本信息

- **洞察链接**: https://news.ycombinator.com/item?id=47633676
- **标题**: Extra usage credit for Claude to celebrate usage bundles launch (Pro, Max, Team)
- **发布时间**: 2小时前
- **分数**: 37 points
- **评论数**: 32 comments
- **来源**: claude.com

## 活动概述

Anthropic 为 Pro、Max 和 Team 订阅用户赠送 $20 额外使用积分，以庆祝 usage bundles 的推出。获取条件为在 2026 年 4 月 3 日上午 9 点（太平洋时间）前订阅相关套餐。

## 用户反馈分析

### 资格获取问题

部分用户反映并未收到赠送积分。推测可能存在未公开的额外条件：之前未获取过额外积分的用户才能领取。有用户指出条款要求"必须启用"额外使用功能，但具体细节仍不明确。

### 营销策略质疑

有用户认为这是诱导用户开启"额外使用"功能的营销手段。开启后如忘记关闭，可能会在不知不觉中消耗额外积分。用户建议使用后应立即关闭自动充值功能。

### 定价体系混乱

评论中指出 Anthropic 的定价体系日益复杂，包含 Free、Pro、Max x5、Max x20、Team、Enterprise 等多个层级，还有会话使用量、每周使用量、额外使用量等多种计费方式，用户对此感到困惑。

### 与 OpenAI 对比

部分用户将 Anthropic 与 OpenAI 进行对比。OpenAI 在过去一个月将订阅使用量乘以 0.5x，且在较便宜的套餐中也能使用长上下文模型。评论认为 Anthropic 可能在容量方面存在问题，有用户反映 Claude Code 请求经常延迟数分钟。

### 积分有效期

赠送积分有效期为 90 天，而常规积分有效期为 12 个月。有用户分享经验称曾在一两个月内消耗了约 $70 CAD 的积分，尤其是使用 Opus 和 Sonnet 4.6 等高级模型时消耗速度极快。

## 总结

用户对此次赠送活动反应不一，有人认为是"不错的 gesture"，也有人质疑 Anthropic 是否面临资金压力或正在流失用户。总体而言，用户对 Anthropic 频繁调整定价和使用政策表示疲劳。
