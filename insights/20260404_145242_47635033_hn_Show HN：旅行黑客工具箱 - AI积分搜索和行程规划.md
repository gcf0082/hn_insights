## 项目简介

**Travel Hacking Toolkit** 是一个让 AI 工具（Claude Code 和 OpenCode）帮助用户进行旅行积分规划和行程安排的工具包。

### 核心功能

用户在使用积分和里程规划旅行时，每次预订都面临同一个决策：使用积分还是付现金？为了回答这个问题，需要查看多个项目的奖励可用性、现金价格、当前余额、转让伙伴比例，以及进行数学比较。作者厌倦了在十几个标签页中手动完成这些操作，因此开发了这个工具包。

该工具包包含：
- **7 个技能**（markdown 文件，包含 API 文档和 curl 示例）
- **6 个 MCP 服务器**（AI 直接调用的实时工具）

### 功能详情

| 功能 | 说明 |
|------|------|
| 奖励航班搜索 | 搜索 25+ 里程计划（通过 Seats.aero） |
| 价格比较 | 比较现金价格（Google Flights、Skiplagged、Kiwi.com、Duffel） |
| 积分余额查询 | 获取用户忠诚度余额（通过 AwardWallet） |
| 酒店搜索 | 搜索酒店（Trivago、LiteAPI、Airbnb、Booking.com） |
| 轮渡路线 | 查找 33 个国家的轮渡路线 |
| 隐藏景点 | 查找目的地附近的隐藏宝藏（通过 Atlas Obscura） |

### 参考数据

工具包包含以下参考数据：
- 转账伙伴比例：Chase UR、Amex MR、Bilt、Capital One、Citi TY
- 积分估值：来自 TPG、Upgraded Points、OMAAT、View From The Wing
- 联盟会员资格、优惠兑换点、预订窗口、酒店连锁品牌查询

### 使用要求

6 个 MCP 服务器中有 5 个无需 API 密钥。只需克隆仓库、运行 setup.sh、即可开始搜索。

## 热门评论摘要

### 1. 扩展功能建议

**yodahuang** 提出了几个扩展建议：
- 添加小型网站如 super.com 和 vio.com 的酒店价格
- 纳入信用卡公司自己的门户，如 Amex FHR 或 Capital One Premier Collection
- 考虑"上半年有 X 积分，所以免费"的情况（如希尔顿度假积分）

作者 borski 回应称已在个人实例中添加了 Chase The Edit 以及 AmEx FHR/HC 酒店，但问题是 AmEx/Chase 没有简单的搜索方式。

### 2. 实用性质疑

**SFO_SIN** 认为这个工具在 10 年前会很有用，但 2026 年的最优策略是：
- "想要第一，买第一"（想要商务舱时付现金）
- "现金为王"——信用卡无脑返现

作者 borski 反驳称自己刚用 140k 积分预订了 8 月去斯堪的纳维亚的商务舱，现金价约 $7k。

### 3. 积分价值讨论

评论中对积分价值进行了激烈讨论：
- **chronic29521**：商务舱通常是单程 100k 积分，140k 积分往返 + $600+ 费用 vs $2900 现金，差别不大
- **SFO_SIN**：现金价 $2,900/人（含费用），8月19日至9月9日（21天），土耳其航空商务舱往返
- **borski**：为 2 人预订，140k 积分 + 约 $1200 费用总共，省了 $4600，很划算

### 4. 其他反馈

- **callumprentice**：作为多年玩积分的人（有百万级 Chase 和 Amex 积分），但有家庭和全职工作，惊讶于积分似乎很不值钱
- **Aboutplants**：剧透警告——积分基本上是骗局
- **hkotcherlakota**：很高兴看到 hype cycle 之外的代理自动化
- **zephyremlin**：Partner 很喜欢旅行积分，这个工具包很棒

## 技术特点

1. **零 API 密钥**：5/6 的 MCP 服务器无需任何 API 密钥
2. **跨平台**：技能是纯 markdown 文件，可在 OpenCode 和 Claude Code 中自动使用
3. **开源协作**：欢迎 PR 扩展工具包
4. **参考数据完整**：包含主流积分计划的转让比例和估值数据

Travel Hacking Toolkit 是一个实用的 AI 旅行规划工具，旨在简化积分兑换的决策过程。虽然社区对积分价值存在争议，但对于拥有大量积分且愿意花时间优化的用户来说，这个工具提供了有价值的自动化帮助。
