## 问题概述

用户在 Windows 系统上使用 Claude Code 登录时遇到 OAuth 超时错误，无法完成登录流程。

## 详细描述

### 问题现象

- Claude Code 在 Windows 上无法登录
- 使用 Google 账号登录时失败
- 错误信息：`OAuth error: timeout of 15000ms exceeded`
- 问题持续存在，多次重试均失败

### 影响范围

- **平台**: Windows
- **终端/Shell**: WSL (Windows Subsystem for Linux)
- **Claude Code 版本**: 2.1.92
- **最后工作版本**: 2.1.92

### 复现步骤

1. 在 Windows 上打开 Claude Code
2. 开始使用 Google 登录
3. 在浏览器中完成登录流程
4. 返回 Claude Code
5. 显示错误：`OAuth error: timeout of 15000ms exceeded`
6. 多次重试登录
7. 每次都出现相同错误

## 问题分析

这是一个 OAuth 认证超时问题，可能的原因包括：

1. **网络问题**：OAuth 回调响应时间超过 15 秒限制
2. **防火墙/代理**：Windows 防火墙或代理阻止了 OAuth 回调
3. **WSL 集成问题**：WSL 环境与 Windows 之间的网络通信问题
4. **认证服务器问题**：Google OAuth 服务响应延迟

## 建议解决方向

1. 检查网络连接和代理设置
2. 确认 Windows 防火墙允许 Claude Code 相关的网络请求
3. 尝试直接使用 Windows 终端而非 WSL 运行 Claude Code
4. 报告问题给 Claude Code 开发团队以便修复
