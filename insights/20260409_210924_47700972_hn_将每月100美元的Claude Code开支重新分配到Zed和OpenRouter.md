## 摘要

本文探讨了用户在Claude Code调整使用限制后，转向 Zed 编辑器配合 OpenRouter 作为替代方案的经验分享。主要讨论了各种AI编程工具的性价比、使用体验以及潜在的替代选择。

---

## 主要观点

### 1. OpenRouter 的优势与不足

**优势：**
- 支持数十种模型，统一API密钥管理
- 可追踪每次请求成本
- 支持在同一请求上运行不同模型并比较结果
- 支持通过不同API密钥分离使用场景
- 不依赖订阅制，可灵活使用API密钥

**不足：**
- OpenRouter 收取 5.5% 的服务费
- Zed 编辑器额外收取 10% 费用
- OpenRouter 支持加密货币支付，导致部分银行拒绝合作

### 2. Claude Code 用户流失

- 有用户反映 Claude Code 每月$100订阅仅能获得约$20-30的实际使用额度
- 配额限制导致使用体验下降
- 多人转向 OpenCode Zen + GitHub Copilot 组合
- Google $20/月计划提供大量 Claude Opus 使用额度

### 3. Pi 编程代理

- Pi 是一个开源编程代理，受到好评
- 可在本地运行，如使用 Qwen3.5-35B 在 Mac M1 Max 上表现良好
- 具备 /tree 功能可回溯历史
- 建议工作流：先让模型探索需求并写入 CONTEXT.md，然后重启会话

### 4. Zed 编辑器体验

- 作为 VSCode 替代品整体评价85%
- 优点：响应速度快，多缓冲区功能实用
- 缺点：内存占用较高（1.2-1.4GB），Linux 上 emoji 渲染有问题
- 扩展系统尚不成熟，Svelte 扩展存在问题

### 5. 替代方案对比

| 方案 | 月费 | 特点 |
|------|------|------|
| OpenRouter + OpenCode | 约$100 | 灵活但有额外费用 |
| GitHub Copilot | $40 | 接入 GPT5/Claude |
| Google AI | $20 | Claude Opus 额度充足 |
| Ollama Cloud | $20 | GLM5 和 Kimi 性价比高 |

---

## 结论

Claude Code 的限制调整促使开发者探索替代方案。OpenRouter 提供了灵活的模型选择，但 5.5% 的服务费需考虑。Zed 作为编辑器获得了较高评价，但扩展生态仍在发展中。Pi 作为开源编程代理展现出潜力。整体而言，开发者需要根据自身工作流程和预算权衡选择。
