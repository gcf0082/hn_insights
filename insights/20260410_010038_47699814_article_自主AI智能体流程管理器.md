# Botctl 洞察报告

## 基本信息

| 项目 | 内容 |
|------|------|
| **项目名称** | Botctl |
| **项目类型** | AI 代理进程管理器 |
| **官方网站** | https://botctl.dev/ |
| **GitHub** | https://github.com/montanaflynn/botctl |
| **支持平台** | macOS / Linux / Windows |

## 项目概述

Botctl 是一个用于管理持久化 AI 机器人的终端仪表盘、Web UI 和声明式配置工具。它允许用户通过 YAML 配置和 Markdown 提示词来创建和管理自主运行的 AI 代理。

## 核心功能

### 1. 声明式配置
- 使用 YAML frontmatter 定义设置
- Markdown body 用于编写提示词
- 配置示例：
  ```yaml
  ---
  name: code-reviewer
  interval_seconds: 60
  max_turns: 20
  ---
  Review open PRs and post comments...
  ```

### 2. 自主执行
- 自动生成 Claude 进程
- 运行、记录日志和休眠循环执行
- 支持后台持续运行

### 3. 会话记忆
- 每次运行保存会话记录
- 支持从中断处继续执行
- 支持向运行中的机器人发送消息重定向

### 4. 热重载
- 修改 BOT.md 后下次运行自动生效
- 无需重启或部署
- 即时更新配置参数

### 5. 可扩展技能
- 从 GitHub 搜索和安装可复用技能模块
- 技能可注入到任何机器人的提示词中
- CLI 命令：
  ```
  botctl skills search slack
  botctl skills add owner/repo --skill slack-notify
  botctl skills list
  ```

### 6. Web 仪表盘
- 浏览器监控和控制机器人
- 支持启动、停止、发送消息和流式日志
- 默认端口 4444，支持自定义端口

## 快速开始

### 安装

**macOS / Linux:**
```bash
curl -fsSL https://botctl.dev/install.sh | sh
```

**Windows:**
```powershell
irm https://botctl.dev/install.ps1 | iex
```

### 创建机器人
```bash
# 交互式创建
botctl create my-bot

# 使用参数创建
botctl create my-bot -d "Monitor weather APIs" -i 300 -m 20
```

### 启动仪表盘
```bash
# 终端仪表盘
botctl

# Web 仪表盘
botctl --web-ui
botctl --web-ui --port 8080
```

### 控制机器人
```bash
# 启动（分离模式）
botctl start my-bot --detach

# 发送消息
botctl start my-bot --message "check the error logs"

# 查看日志
botctl logs my-bot -f

# 停止
botctl stop my-bot
```
