## 项目概述

BrightBean Studio 是一个开源、可自托管的社交媒体管理平台，专为创作者、代理机构和中型企业设计。它是 Sendible、SocialPilot 或 ContentStudio 的免费替代方案，允许用户从一个统一的多工作区仪表板规划、撰写、调度、审批、发布和监控跨 Facebook、Instagram、LinkedIn、TikTok、YouTube、Pinterest、Threads、Bluesky、Google Business Profile 和 Mastodon 等 10+ 平台的内容。

**核心定位**: 帮助管理多个客户账户的团队掌控自己的社交媒体栈，而非每月向 SaaS 供应商支付 100-300 美元。

---

## 主要功能

### 多工作区与团队
- 无限组织 → 工作区 → 成员结构
- 精细的 RBAC 权限控制，支持自定义角色和邀请机制
- 独立的客户端角色供外部协作者使用

### 内容编辑器
- 富文本编辑器，支持每个平台的标题/媒体覆盖
- 版本历史、可复用模板、内容分类与标签
- Kanban 创意看板

### 日历与排期
- 可视化日历，支持每个账户的重复每周发布时段
- 命名队列自动将帖子分配到下一个可用时段

### 发布引擎
- 直接的一级方 API 集成（无聚合器中间商）
- 自动重试机制，按账户追踪速率限制
- 90 天发布审计日志

### 审批工作流
- 可配置阶段（无/可选/内部/内部+客户端）
- 线程化内部与外部评论、提醒功能
- 完整审计跟踪

### 统一社交收件箱
- 整合来自所有已连接平台的评论、提及、私信和评价
- 情感分析、任务分配、线程化回复
- 历史数据回填

### 媒体库
- 按组织和工作区作用域划分的媒体库
- 嵌套文件夹，自动生成平台优化变体
- Alt 文本管理

### 客户端门户
- 无需密码的 30 天魔法链接访问
- 客户端可直接审批或拒绝帖子

### 通知系统
- 应用内、邮件和 Webhook 投递
- 每用户偏好设置覆盖所有事件类型

### 安全与运维
- 加密的令牌与凭证存储
- 可选 2FA（TOTP）
- Google/GitHub SSO 支持
- Sentry 监控集成
- 7 天可逆的组织删除宽限期

### 白标友好
- 按工作区定制品牌（标志、颜色）
- 工作区默认设置：标签、首发评论、发布模板

## 总结

BrightBean Studio 是一个功能完整的开源社交媒体管理解决方案，其核心优势在于：

1. **零成本无限制** - 所有功能对所有用户开放，无付费层级、无功能门控
2. **真正自托管** - 数据完全由用户掌控，避免供应商锁定
3. **直接 API 连接** - 绕过聚合器，直接对接各平台官方 API
4. **完整功能覆盖** - 从内容创作到发布管理再到数据分析，提供一体化工作流

对于需要管理多个客户账户但不想支付高昂 SaaS 费用的团队而言，这是一个极具吸引力的开源替代方案。
