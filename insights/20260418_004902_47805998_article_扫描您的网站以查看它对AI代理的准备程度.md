# 洞察报告：Is Your Site Agent-Ready?

**洞察链接**: https://isitagentready.com

**基本信息**:
- **网站名称**: Is Your Site Agent-Ready?
- **功能**: 扫描网站以检查其是否为AI代理（AI Agents）做好准备
- **检查标准**: llms.txt、MCP、代理技能（Agent Skills）以及其他代理友好标准

---

## 网站概述

Is Your Site Agent-Ready? 是一个由 Cloudflare 提供的免费工具，用于评估网站对AI代理的适配程度。该工具通过多项新兴标准对网站进行全面检查，帮助网站运营者了解其网站在AI代理时代的准备情况。

## 检查项目

网站提供多种检查类别，涵盖五个主要领域：

### 1. 可发现性（Discoverability）
- robots.txt 文件
- 网站地图（Sitemap）
- 链接响应头（Link headers）

### 2. 内容可访问性（Content Accessibility）
- Markdown 内容协商（Markdown negotiation）

### 3. 机器人访问控制（Bot Access Control）
- AI 机器人规则（AI bot rules）
- 内容信号（Content Signals）
- Web 机器人认证（Web Bot Auth）

### 4. 协议发现（Protocol Discovery）
- MCP 服务器卡片（MCP Server Card）
- 代理技能（Agent Skills）
- WebMCP
- API 目录（API Catalog）
- OAuth 发现（OAuth discovery）
- OAuth 保护资源（OAuth Protected Resource）
- A2A 代理卡片（Agent Card）

### 5. 商务（Commerce）
- x402（微支付协议）
- UCP（统一支付协议）
- ACP（代理商务协议）

## 如何使用

1. 输入要扫描的网站URL
2. 选择扫描类型：全部、内容站点或API/应用
3. 自定义检查项目
4. 获取详细的评估报告

## 提升评分的建议

### 简单易行的改进措施：

1. **发布有效的 robots.txt** - 包含 AI 机器人规则和网站地图指令
2. **确保首页包含有用的发现头信息或元数据**

### 深入学习：

访问 [Cloudflare Agents 文档](https://developers.cloudflare.com/agents/) 了解如何构建和部署能够浏览、交互和交易的 AI 代理。
