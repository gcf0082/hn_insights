## 项目简介

Ctx 是一个本地 SQLite 支持的 skill，适用于 Claude Code 和 Codex。它将上下文存储为持久化的工作流（workstream），可以在不同的 agent 会话之间继续使用。每个工作流可以包含多个会话、笔记、决策、待办事项和恢复包。它的核心功能是在不同编程 agent 之间提供一个可迁移的 /resume 方案。

## 核心功能

1. **跨工具恢复工作流** - 可以在 Claude 和 Codex 之间无缝切换并恢复之前的工作状态
2. **上下文拉取** - 将现有上下文拉取到新工作流中继续使用
3. **稳定的会话绑定** - 确保工作流与原始会话文件保持绑定，不会漂移到最新的会话
4. **工作流搜索** - 跨历史工作流搜索相关内容
5. **分支功能** - 从现有上下文分支，探索并行任务

## 技术特点

- **本地优先设计**：使用 SQLite 存储，无 API 密钥需求，无托管后端
- **一键安装**：通过 `./setup.sh` 即可同时添加到 Claude Code 和 Codex
- **跨平台兼容**：同时支持 Claude Code（使用 `/ctx [command]`）和 Codex（使用 `ctx [command]`）
