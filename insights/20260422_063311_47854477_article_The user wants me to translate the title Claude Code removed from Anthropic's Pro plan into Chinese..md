## 定价概览

Claude 提供个人版和团队/企业版两种主要定价模式，同时提供 API 接口服务。

### 个人用户定价

| 计划 | 价格 | 主要特点 |
|------|------|----------|
| **Free** | $0/月 | 免费使用，包含基础聊天功能、代码生成、数据可视化、写作编辑、内容分析、图片分析、网络搜索等 |
| **Pro** | $17/月（年付）<br>$20/月（月付） | 更多使用额度，包含 Claude Cowork、无限制项目、研究功能、更多模型访问权限 |
| **Max 5x** | $100/月起 | Pro 所有功能 + 5倍使用额度，高输出限制 |
| **Max 20x** | 定价咨询 | Pro 所有功能 + 20倍使用额度，优先访问权限 |

### 团队定价（5-150人）

| 座位类型 | 价格 | 特点 |
|----------|------|------|
| **标准座位** | $20/人/月（年付）<br>$25/人/月（月付） | 所有 Claude 功能 + 更多使用额度 |
| **Premium 座位** | $100/人/月（年付）<br>$125/人/月（月付） | 5倍标准座位使用额度 |

团队计划包含：Claude Code、Claude Cowork、Microsoft 365/Slack 集成、企业搜索、集中计费管理、SSO 域名验证等。

### 企业定价

- **价格**: $20/座位 + API 费率使用费
- **特点**: 可设置用户和组织消费限额、Google Docs 编目、基于角色的细粒度权限管理、SCIM 审计日志、合规 API、自定义数据保留控制、网络级访问控制、IP 白名单、HIPAA 合规版本

## 核心洞察

1. **分层定价策略**: Claude 采用清晰的分层定价，从免费个人版到企业版，满足不同规模用户需求
2. **API 成本优势**: Haiku 模型输入仅 $1/MTok，输出 $5/MTok，是成本效益最高的选择
3. **批处理节省**: 批处理可节省 50% 成本，适合大规模异步处理场景
4. **团队协作强化**: 团队计划强调企业级安全和管理功能（SSO、SCIM、审计日志等）
5. **教育优惠**: 提供针对高校的综合性教育计划，包括学生和教职员工折扣
