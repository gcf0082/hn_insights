## 摘要

Anthropic 宣布不再向新用户提供每月20美元的 Claude Code Pro 订阅服务。该变化引发了 Hacker News 社区的广泛讨论，用户对 AI 订阅服务的未来和定价策略表达了担忧。

## 主要讨论观点

### 1. 依赖云订阅的风险
- 用户警告不应将自己的创造力完全寄托于订阅服务
- 建议始终保持本地模型作为备选方案
- 产品可能随时改变，旧版本无法继续使用

### 2. 定价策略讨论
- 有用户认为可以参照标准 API 定价来判断 AI 产品成本，而非促销订阅价格
- 认为价格仍然受到投资者补贴的影响
- 被比作"赌场"——用户为 token 投入资金期望获得回报

### 3. 本地模型的可行性
- 关于本地模型是否能与 Claude Opus 4.6 性能相匹配的讨论
- 主流开发者机器 RAM 通常为 16GB，难以运行大型模型
- Qwen3.5 和 Gemma 等约30B 模型被认为是可行的折中方案

### 4. 供需关系
- 有评论指出这是简单的供需关系决定的价格变化
- 更高的需求意味着价格上涨，这在计算机时代之前就已存在

### 5. 市场竞争格局
- 部分用户认为 Anthropic 在政府压力下已失去对公司的控制
- 预测 Google 可能是最终赢家
