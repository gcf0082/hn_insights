# OpenAI 推出 ChatGPT 工作区智能体：AI 赋能团队协作自动化

**洞察链接**: https://openai.com/index/introducing-workspace-agents-in-chatgpt/

**发布日期**: 2026年4月22日

**来源**: OpenAI 官方博客

---

## 核心要点

OpenAI 正式发布 ChatGPT 工作区智能体（Workspace Agents），这是一款基于 Codex 技术构建的团队协作自动化平台，标志着 ChatGPT 从个人对话工具向企业级工作自动化平台的重大转型。

## 产品定位

工作区智能体是 GPTs（自定义 GPT）的进化版本，专为团队设计而非个人用户。它们可以：

- 从正确的系统中获取上下文信息
- 遵循团队工作流程
- 在需要时请求审批
- 跨工具保持工作推进
- 离线运行时也能持续工作

这些智能体可在组织内共享，维护团队知识资产，并配备企业级监控和控制功能以保护敏感数据。

## 核心能力

1. **多任务处理**：能够执行准备报告、编写代码、回复消息等多种工作任务
2. **云端运行**：在云端运行，即使用户离线也能持续工作
3. **跨工具集成**：支持在 ChatGPT 和 Slack 中与智能体交互
4. **模板支持**：提供金融、销售、营销等领域的预置模板
5. **权限控制**：在组织设定的权限和控制范围内运行

## 可用性与定价

- 即日起通过 ChatGPT Business、Enterprise、Edu 和 Teachers 计划提供研究预览
- 未来将支持将自定义 GPT 直接转换为工作区智能体

## 技术架构

工作区智能体由 Codex 驱动，每个智能体拥有独立的工作区，可访问：

- 文件
- 代码
- 工具
- 记忆功能

## 行业影响

这是 OpenAI 在AI智能体领域的重大布局，直面 Anthropic 的 Claude Coworker 等竞争对手。企业级智能体市场战局正在升温。
