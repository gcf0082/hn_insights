## 技术概述

OpenAI 发布了 Privacy Filter，一个用于检测和脱敏文本中个人身份信息（PII）的开源模型。该模型采用双向 token 分类架构，通过 span 解码方式实现高效的个人信息识别。

## 核心参数

| 指标 | 数值 |
|------|------|
| 总参数 | 1.5B |
| 活跃参数 | 50M |
| 上下文长度 | 128K tokens |
| 许可证 | Apache 2.0 |

## 支持的PII类型（8种）

1. 姓名 (Names)
2. 地址 (Addresses)
3. 邮箱 (Email addresses)
4. 电话 (Phone numbers)
5. URL链接
6. 日期 (Dates)
7. 账号 (Account numbers)
8. 密钥 (Passwords, API keys)

## 技术架构

- **模型类型**: 双向 token 分类器 (bidirectional token-classification model)
- **解码方式**: 受约束的 Viterbi 过程 (constrained Viterbi procedure)
- **工作方式**: 一次性标注整个输入序列，然后解码连贯的 span
- **推理方式**: 从预训练语言模型 checkpoint adapt 到 token 分类器

## 性能表现

- 标准基准测试 (PII-Masking-300k): **96%**
- 修正版本: **97.43%**
