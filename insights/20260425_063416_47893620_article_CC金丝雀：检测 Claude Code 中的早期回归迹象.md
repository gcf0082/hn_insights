## 项目简介

cc-canary 是一款面向 Claude Code 的**漂移检测工具**，以 Agent Skills 的形式封装。它能够读取 Claude Code 已有的 JSONL 会话日志，分析模型是否在用户自己的工作上出现漂移，并生成可分享的取证报告。

该工具的核心理念是**完全本地化运行**，无需网络连接、无需账户、无需遥测数据、也无需后台守护进程，仅基于本地磁盘已有的数据进行检测。

## 核心功能

cc-canary 提供两个技能：

1. **cc-canary**: 执行取证分析并生成 Markdown 格式报告
2. **cc-canary-html**: 生成深色主题 HTML 仪表板报告，并自动在浏览器中打开

报告包含以下内容：

- **判定结果**: HOLDING / SUSPECTED REGRESSION / CONFIRMED REGRESSION / INCONCLUSIVE
- **主要指标表**: 对比前后指标，包含可视化状态指示（🟢🟡🔴）
- **周趋势图**: 成本、读写比率、推理循环、令牌数等
- **跨版本对比**: 同一用户在不同模型版本下的表现
- **自动检测拐点**: 综合健康评分突变日期
- **详细发现**: 模型侧/用户侧/模糊分类的问题分析

## 技术实现

cc-canary 通过以下步骤工作：

1. **扫描**: Python 脚本遍历 `~/.claude/projects/**/*.jsonl`，按时间窗口过滤
2. **去重**: 基于 `(message.id, requestId)` 进行消息去重
3. **聚合**: 计算各项会话指标
4. **拐点检测**: 基于综合健康评分检测异常日期
5. **预渲染**: 生成 Markdown/HTML 骨架
6. **填充**: 由 Claude 填写叙事内容并保存

运行时长约 2.5 秒脚本执行 + 10-20 秒叙事填充。

## 追踪指标

核心追踪指标包括：

- **读写比率**: 每次文件编辑前的读取次数
- **写入占比**: Write 在突变中的比例
- **推理循环**: 推理中的循环短语频率
- **挫折率**: 用户提示中表达挫折的比率
- **思维编辑率**: 被编辑的思维块比例
- **平均思维长度**: 推理深度代理
- **每用户回合的 API 调用次数**
- **每用户回合的令牌数**

## 隐私特性

- 完全本地运行，零网络调用
- 仅读取指定的 JSONL 文件
- 用户提示内容截断至 180 字符
- 自动脱敏路径、邮箱、令牌等信息

## 安装方式

```bash
npx skills add delta-hq/cc-canary
```

使用示例：

```bash
/cc-canary 60d
/cc-canary-html 30d
```

时间窗口支持：7d / 14d / 30d / 60d / 90d / 180d

## 适用场景

cc-canary 适合以下用户：

- 需要监控 Claude Code 行为变化的用户
- 关注模型性能稳定性的开发者
- 需要生成模型问题取证报告的用户
- 注重隐私且希望本地检测的用户
