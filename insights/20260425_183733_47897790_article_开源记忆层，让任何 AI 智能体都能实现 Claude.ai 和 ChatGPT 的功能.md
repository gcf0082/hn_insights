## 项目概述

Stash 是一个开源的持久认知层，旨在为 AI 智能体提供长期记忆能力。它位于 AI 智能体与世界之间，不替换模型，而是使模型具有连续性。项目采用 PostgreSQL + pgvector 作为底层存储，支持 MCP 协议，可与任何 MCP 兼容的智能体集成。

**核心特性**:
- 28 个 MCP 工具
- 6 个管道处理阶段
- 支持无限数量的智能体
- Apache 2.0 开源许可证

---

## 核心功能

### 1. 持久记忆系统

Stash 解决了 AI 智能体的"健忘症"问题。在没有 Stash 的情况下，用户每次新会话都需要重新解释项目背景；而有了 Stash 后，智能体会记住之前的对话内容，从上次中断的地方继续工作。

**对比效果**:
- 无 Stash: 每次新会话都问"你是谁？"，需要重新解释所有内容
- 有 Stash: 自动从上次结束的地方继续，记住用户的偏好和目标

### 2. 命名空间管理

Stash 采用类似文件夹的命名空间组织方式，不同类型的记忆存储在不同的命名空间中：

```
/users/alice        - 用户信息和偏好
/projects          - 所有项目
  /projects/restaurant-saas  - 餐饮 SaaS 项目
  /projects/mobile-app       - 移动应用项目
/self               - 智能体自我认知
  /self/capabilities       - 智能体擅长什么
  /self/limits            - 智能体的局限性
  /self/preferences       - 智能体的工作偏好
```

### 3. 记忆整合管道

Stash 提供了一个后台处理管道，将原始记忆逐步转化为结构化知识：

1. ** Episodes（片段）**: 存储原始观察记录
2. ** Facts（事实）**: 由 LLM 合成的结构化信念
3. ** Relationships（关系）**: 从事实中提取的实体关系图
4. ** Causal Links（因果链）**: 事实之间的因果关系
5. ** Patterns（模式）**: 更高层次的抽象洞察
6. ** Contradictions（矛盾）**: 自我纠正和置信度衰减
7. ** Goal Inference（目标推理）**: 自动跟踪目标进度
8. ** Failure Patterns（失败模式）**: 检测重复错误
9. ** Hypothesis Scan（假设扫描）**: 被动确认或拒绝开放假设

## 与 RAG 的对比

| 特性 | RAG | Stash |
|------|-----|-------|
| 知识来源 | 仅限提供的文档 | 从所有对话中学习 |
| 学习能力 | 无法从对话中学习 | 自动从每次对话中学习 |
| 目标跟踪 | 无法跟踪目标或意图 | 跨周跨月跟踪目标 |
| 因果推理 | 无法推理因果关系 | 支持因果推理 |
| 状态连续性 | 无状态，每次从头开始 | 连续状态，从上次结束的地方继续 |
| 知识创建 | 必须先手动编写知识 | 自动创建知识 |

**简言之**: RAG 就像一个高效的图书管理员——能完美读取文件，但离开房间就会忘记一切。Stash 则像一位从第一天就在场的同事，记住你做过的每个决定，并且每周都变得更有价值。

## 快速开始

只需要 3 条命令即可运行 Stash：

```bash
# 1. 克隆仓库
$ git clone https://github.com/alash3al/stash
$ cd stash

# 2. 配置环境
$ cp .env.example .env
# 编辑 .env 设置 API 密钥和模型偏好

# 3. 启动所有服务
$ docker compose up
```

**注意**: 首次运行前必须在 `.env` 中设置 `STASH_VECTOR_DIM`，初始化后无法更改。

## 技术架构

- **本地模型**: Ollama（Qwen、Llama、Mistral 等）

## 总结

Stash 是一个革命性的开源项目，它为 AI 智能体提供了真正的持久记忆能力。与传统的 RAG 系统和其他闭源记忆方案不同，Stash 具有以下优势：

1. **模型无关性**: 支持任何 AI 模型，不受供应商锁定
2. **开源免费**: Apache 2.0 许可证，完全透明
3. **深度学习**: 不仅记住事实，还学习模式、跟踪目标、从失败中学习
4. **用户数据拥有权**: 数据完全由用户控制
5. **自我认知能力**: 智能体可以了解自己的能力、局限性和工作偏好

对于需要长期上下文和持续学习的 AI 应用场景，Stash 提供了一个强大而灵活的解决方案。
