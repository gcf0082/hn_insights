## 三种配置方式

### 1. 最简便方法：使用 /model 命令

1. 启动 Claude Code：`claude`
2. 输入 `/model` 并从交互式菜单中选择 desired 模型
3. 更改立即生效

> 可随时通过 `/status` 命令查看当前使用的模型

### 2. 仅当前会话有效：--model 标志

```bash
# Opus 4.7
claude --model claude-opus-4-7

# Sonnet 4.6
claude --model claude-sonnet-4-6

# Opus 4.6
claude --model claude-opus-4-6

# Opus 4.5
claude --model claude-opus-4-5-20251101

# Haiku 4.5
claude --model claude-haiku-4-5-20251001

# Sonnet 4.5
claude --model claude-sonnet-4-5-20250929
```

### 3. 永久默认配置：环境变量

**步骤1**：确定 shell 类型
```bash
echo $SHELL
# /bin/zsh → macOS
# /bin/bash → Linux
```

**步骤2**：添加到 shell 配置文件

对于 ZSH 用户（macOS）：
```bash
echo 'export ANTHROPIC_MODEL="claude-opus-4-7"' >> ~/.zshrc
```

对于 BASH 用户（Linux）：
```bash
echo 'export ANTHROPIC_MODEL="claude-opus-4-7"' >> ~/.bashrc
```

**步骤3**：应用更改
```bash
# ZSH
source ~/.zshrc

# BASH
source ~/.bashrc
```

**步骤4**：关闭并重新打开终端

**步骤5**：启动 Claude Code：`claude`

---

## 总结

Claude Code 提供了灵活的三层模型配置方式：
1. `/model` 命令适合快速临时切换
2. `--model` 标志适合单次会话使用特定模型
3. 环境变量适合设置永久默认模型

用户可根据实际需求选择最适合的配置方式。
