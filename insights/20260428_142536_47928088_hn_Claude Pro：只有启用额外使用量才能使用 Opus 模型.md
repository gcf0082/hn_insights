## 核心内容

Anthropic 正在对其帮助支持页面进行更改，涉及 Pro 计划用户如何使用 Opus 模型的新定价策略。根据当前官方页面的说明，使用 Claude Code 的 Pro 计划用户只能在启用并购买额外使用量后才能访问 Opus 模型。

### 定价争议

用户分享了自己的使用数据：$200 月费计划在过去30天内使用了3000万 tokens，按此计算相当于约$1500。即使企业获得50%折扣，费用仍然过高。一般用户的使用量在 $200-$400 范围内是常见情况。

## 社区观点

### 批评与担忧

- **文档混乱**: 社区普遍批评 Anthropic 的文档更新不及时，支持页面的信息经常前后不一致
- **定价过高**: 有评论指出 "vibe-coded documentation" 质量堪忧，且付费模式正在变得难以承受
- **模型性能下降**: 用户反映订阅计划中的模型存在"暗箱操作"现象，性能不如从前

### 市场趋势讨论

- **AI 依赖风险**: 社区讨论了企业对 AI 工具的依赖正在加深，形成了类似"上瘾"的局面
- **开源替代方案**: 有评论指出中国开发的开源模型正在崛起，可能成为替代选择
- **IPO 影响**: 有人提到 OpenAI 和 Anthropic 都计划今年上市，在财务压力下可能需要更加注重盈利

### 技术讨论

- **思考模式调整**: 有用户发现将思考设置为 "high" 而非 "x-high" 可获得相当质量但减少 tokens 消耗
- **自我托管**: 社区认为开源模型在推理优化硬件上运行可能是长期更经济的选择

## 结论

此次定价调整反映了 AI 行业面临的商业化挑战。用户对不断提高的成本和混乱的文档表示不满，同时开始关注开源替代方案。随着主要 AI 公司筹备上市，定价策略和技术服务的平衡将成为关键问题。
