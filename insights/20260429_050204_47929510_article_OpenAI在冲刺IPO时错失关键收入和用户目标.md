# OpenAI 收入与用户目标未达成：IPO前景蒙阴影

## 基本信息

- **洞察链接**: https://www.wsj.com/tech/ai/openai-misses-key-revenue-user-targets-in-high-stakes-sprint-toward-ipo-94a95273
- **发布日期**: 2026年4月28日
- **来源**: Wall Street Journal
- **主题**: OpenAI 营收与用户增长未达内部目标，IPO计划存疑

## 核心发现

OpenAI 在近几个月未能达成其用户获取和收入目标，引发公司内部对是否能承担大规模数据中心支出的担忧。

### 关键数据

- **用户目标未达成**: OpenAI 未能实现年内让 ChatGPT 达到每周10亿活跃用户的目标
- **收入增长放缓**: 今年多个月度收入目标未能完成
- **增长放缓**: 每月用户增长从年初的42%下降到9月份的13%
- **实际营收**: 2025年实际确认收入约131亿美元，与年度经常性收入(ARR)目标存在差距

### 竞争压力

- 在编码和企业市场流失份额给竞争对手 Anthropic
- 面临订阅用户流失问题

### 财务担忧

CFO Sarah Friar 向其他公司领导层表达担忧：如果收入增长不够快，可能无法为未来的计算合同提供资金。公司每产生1美元收入就要花费约2.25美元。

### 战略布局

- Oracle 与 OpenAI 签署了为期五年、价值3000亿美元的计算协议
- Nvidia 已向这家初创公司承诺数十亿美元
- OpenAI 与 Amazon 达成重大战略合作，将现有380亿美元支出协议扩展100亿美元

## 影响分析

1. **IPO不确定性增加**: 财务压力和增长放缓可能影响原计划于今年晚些时候进行的IPO
2. **成本控制收紧**: 董事会正更密切地审查计算协议，Friar 正在与高管团队设法控制成本
3. **支出与收入失衡**: 高昂的数据中心投资与有限的收入增长形成矛盾

## 总结

OpenAI 在经历2026年2月宣布的1100亿美元创纪录融资后，面临着增长放缓和盈利能力挑战。该公司能否成功推进IPO并支撑其大规模数据中心支出计划，仍存在重大疑问。
