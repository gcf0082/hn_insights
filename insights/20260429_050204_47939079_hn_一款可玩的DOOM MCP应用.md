## 洞察总结

这是一个创新的实验，展示了如何在 ChatGPT 和 Claude 等 AI 聊天界面中运行可玩的 DOOM 游戏。该项目利用 MCP（Model Context Protocol）协议，将 DOOM 游戏作为嵌入式资源加载到聊天窗口中。

### 技术实现

DOOM 实际上并非在 AI 模型本身运行，而是通过以下方式实现：
- 使用 MCP 资源注册机制
- 通过 iframe 在聊天界面中渲染远程网页
- 游戏实际上运行在 Cloudflare 的 doom-wasm 上

### 社区讨论要点

1. **MCP 潜力**: 评论者认为 MCP 作为应用平台的潜力被低估了不仅仅可以连接 Google Drive 等服务，还能支持聊天内的实际应用

2. **标题争议**: 有评论指出原标题"DOOM runs in ChatGPT and Claude"有误导性，实际是"in"而不是"on"AI——游戏运行在外部服务器上，AI只是观察者

3. **类似项目**: 有开发者提到在 Open WebUI 中也进行过 Snake 和 DOOM 的测试，游戏是测试新技术的很好方式

4. **未来展望**: 有人思考"Can it run Doom?"这个经典测试未来可能会变成"Can it run AGI?"

### 意义

这个项目代表了 AI 聊天应用的新可能性，展示了未来 AI 助手可能不仅仅是对话界面，还可以成为各种应用的承载平台。MCP 协议为此类集成提供了技术基础。
