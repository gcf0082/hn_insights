# Anthropic 发布 Claude for Creative Work：AI 赋能创意工作者

**洞察链接**: https://www.anthropic.com/news/claude-for-creative-work
**发布日期**: 2026年4月28日
**来源**: Anthropic 官方新闻

---

## 核心摘要

Anthropic 今日宣布推出一系列与创意工具的连接器（Connectors），使 Claude 能够与专业创意软件协同工作。这些连接器由包括 Blender、Autodesk、Adobe、Ableton 和 Splice 在内的合作伙伴联盟共同开发，让创意专业人士能够扩展自己的能力边界，更快速地进行更大规模的项目创作。

---

## 主要连接器功能

### 新增的创意工具连接器

| 连接器 | 功能描述 |
|--------|----------|
| **Ableton** | 基于 Live 和 Push 官方产品文档提供精准回答 |
| **Adobe** | 支持从 Creative Cloud 50+ 工具（包括 Photoshop、Premiere、Express 等）调用图像、视频和设计 |
| **Affinity by Canva** | 自动执行批量图像调整、图层重命名、文件导出等重复性生产任务 |
| **Autodesk Fusion** | 允许订阅用户通过对话创建和修改 3D 模型 |
| **Blender** | 提供自然语言接口访问 Python API，便于探索和理解复杂设置 |
| **Resolume Arena/Wire** | 允许 VJ 和现场视觉艺术家通过自然语言实时控制演出 |
| **SketchUp** | 将与 Claude 的对话转化为 3D 建模起点 |
| **Splice** | 允许音乐制作人在 Claude 内搜索无版税样本目录 |

## Claude 在创意工作中的应用场景

1. **学习和掌握创意工具**：Claude 可充当复杂软件的按需导师，解释修饰符堆栈、演示合成技术或展示新功能使用方法。

2. **用代码扩展工具**：Claude Code 可以为常用软件编写脚本、插件和生成系统，如自定义着色器、程序化动画或参数化模型。

3. **桥接工具管线**：Claude 可以在不同应用间转换格式、重构数据并保持资产同步，实现设计、3D 和音频工具间的无缝衔接。

4. **快速探索和交接**：Claude Design 可用于探索软件体验创意，可视化选项并根据反馈迭代，支持导出到 Canva 等工具。

5. **处理重复性生产工作**：Claude 可处理批量资源处理、项目脚手架搭建或场景程序化修改等步骤，减少繁琐工作。

## 与 Blender 的深度合作

Blender 是跨行业使用的免费开源 3D 创建套件，涵盖独立游戏开发、动态图形、建筑可视化和电影制作等领域。Blender 开发者创建的 MCP 连接器已正式可用，3D 艺术家可以使用 Blender 连接器分析和调试整个场景，或构建自定义脚本批量修改场景中的对象。

Anthropic 已加入 Blender Development Fund 成为赞助者，支持其继续开发 Python API，使此类集成成为可能。

---

## 教育合作计划

Anthropic 还与艺术和设计项目合作，支持涉及创意计算的课程。首批合作项目包括：

- 罗德岛设计学院艺术与计算专业
- 林林艺术设计学院创意 AI 基础课程
- 伦敦大学金匠学院计算艺术硕士/文学硕士项目

学生和教师将获得 Claude 和新连接器的访问权限，其反馈将帮助了解创意从业者的需求。

## 行业影响

该发布标志着 AI 工具与专业创意工作流程的深度整合，通过 MCP 协议实现的开放互操作性，将为创意行业带来新的工作方式的可能性。
