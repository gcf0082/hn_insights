## 概述

Mike 是一个开源法律人工智能平台，旨在成为 Harvey 和 Legora 的开源替代方案。

## 核心功能

### 1. 助手（Assistant）
一个聊天界面，可以读取文档、引用原文、运行多步骤工作流，并端到端地起草和编辑合同。用户可以插入自己的 Claude 或 Gemini 密钥，完全控制所使用的模型。

### 2. 项目（Projects）
以事项为范围的工作空间。用户可以将信用协议、 SPA、租赁和尽职调查资料包上传到项目中，助手可以在每次对话和每个文档中保持完整的上下文。

### 3. 表格审查（Tabular Review）
跨数百个文档并行的电子表格式提取。每个单元格都可以追溯到具体页码和引用，没有 hallucinated 答案或死链接。

### 4. 工作流（Workflows）
将经过验证的提示保存为可重用的工作流，包括 CP 信用协议摘要和控制权变更审查清单。创建可在 Firm 范围内一键运行的模板。

## 为什么选择开源

- **零许可费用**: 无供应商锁定，无每年上涨的按座位定价，只需支付模型 API 费用
- **可自托管**: 部署在自己的基础设施内，文档永远不会离开您的网络边界
- **可分支和扩展**: 添加特定实践的工作流，集成 DMS，接入自己的引用引擎
- **可审计**: 对提示构建、引用解析和数据流的方式进行全面审计

## 起步方式

用户可以从云端开始使用，也可以从 GitHub 克隆代码库自行部署。
