# HomeSec-Bench — 本地 AI vs 云端基准测试洞察

## 核心发现

**Qwen3.5-9B 在 MacBook Pro M5 上本地运行，得分 93.8%**，与 GPT-5.4（97.9%）仅相差 4.1 个百分点，与 GPT-5.4-mini（95.8%）相差 2 个百分点。它甚至比 GPT-5.4-nano 高出 1 个百分点。

关键性能指标：
- **解码速度**: 25 tok/s
- **首 Token 时间 (TTFT)**: 765ms
- **GPU 内存**: 13.8 GB 统一内存
- **完全离线**: 零 API 成本，完全数据隐私

## 性能对比

### 首 Token 时间 (TTFT) — 越低越好
- **Qwen3.5-35B-MoE: 435ms** — 比所有 OpenAI 云模型都快
- GPT-5.4-nano: 508ms
- GPT-5.4-mini: 553ms
- GPT-5.4: 601ms
- Qwen3.5-9B: 765ms

## 测试覆盖范围

HomeSec-Bench 评估 LLM 在真实家庭安全助手工作流中的表现，涵盖 16 个测试套件：

1. **工具使用** (16 测试) — 正确选择工具及参数
2. **安全分类** (12 测试) — Normal → Monitor → Suspicious → Critical 分类
3. **聊天与 JSON 合规** (11 测试) — 人设、JSON 输出、多语言
4. **事件去重** (8 测试) — 跨摄像头识别同一人
5. **上下文预处理** (6 测试) — 去重对话、保留系统消息
6. **VLM 到告警分类** (5 测试) — 端到端：VLM 输出 → 紧急程度 → 告警分发
7. **知识注入** (5 测试) — 使用注入知识个性化响应
8. **告警路由** (5 测试) — 通道路由、勿扰时间解析
9. **提示注入抵抗** (4 测试) — 角色混淆、提示提取、升级
10. **多轮推理** (4 测试) — 引用解析、时间跨度
11. **错误恢复** (4 测试) — 处理不可能的查询、API 错误
12. **隐私与合规** (3 测试) — PII 脱敏、拒绝非法监控
13. **主题分类** (4 测试) — 将查询路由到正确领域
14. **知识蒸馏** (5 测试) — 从对话中提取持久事实
15. **叙事综合** (4 测试) — 将事件日志汇总为每日报告

## 关键洞察

1. **本地 AI 性能已接近云端**: Qwen3.5-9B 在消费级硬件（MacBook Pro M5）上达到 93.8% 通过率，与顶级云端模型差距极小。

2. **性价比优势明显**: 本地运行完全零 API 成本，数据永不离开设备。

3. **小模型高效**: Qwen3.5-9B 仅用 13.8GB 内存达到最佳本地性能，是资源受限场景的理想选择。

4. **MoE 模型 TTFT 领先**: Qwen3.5-35B-MoE 的首 Token 时间（435ms）比所有 OpenAI 云模型都快。

5. **针对性强**: 该基准测试专注于家庭安全领域的实际任务，而非通用聊天。

## 系统信息

- **测试系统**: Aegis-AI — 本地优先 AI 家庭安全
- **运行设备**: MacBook Pro M5 · M5 Pro · 18 核心 · 64GB 统一内存 · macOS 15.3 (arm64)
- **推理引擎**: llama.cpp
- **测试图像**: 35 张 AI 生成图像（无真实用户素材）
