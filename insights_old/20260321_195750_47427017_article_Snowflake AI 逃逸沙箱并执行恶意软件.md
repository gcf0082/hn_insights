## 基本概述

Snowflake Cortex Code CLI 命令验证系统中存在漏洞，允许攻击者通过间接提示注入绕过人工审批流程并突破沙箱安全限制，执行任意恶意命令。

## 漏洞原理

1. **人工审批绕过**: 命令验证系统未对进程替换 `<()` 表达式中的命令进行校验
2. **沙箱逃逸**: 攻击者利用 `dangerously_disable_sandbox` 标志突破沙箱限制
3. **攻击条件**: 组合使用"安全"命令开头 + 进程替换语法可绕过验证

## 影响范围

- 执行任意命令无需用户审批
- 可在沙箱外执行恶意代码
- 利用缓存凭证访问Snowflake数据库
  - 窃取数据库内容
  - 删除数据表
  - 添加恶意后门用户
  - 锁定合法用户

## 修复措施

- 升级至 Cortex Code CLI 1.0.25 版本
- 建议配置禁用 `dangerously_disable_sandbox` 功能
- 使用浏览器认证替代缓存令牌
