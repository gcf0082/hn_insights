## 概述

本速查表是 Claude Code v2.1.81 的完整参考指南，最后更新于 2026 年 3 月 23 日。Claude Code 是 Anthropic 推出的 AI 编程助手 CLI 工具，本速查表涵盖了从基础操作到高级功能的完整命令和技巧。

---

## 核心功能更新

### 新增特性
- `--bare` 标志 — 极简无头模式（无钩子/LSP/插件）
- `--channels` — 权限中继和 MCP 推送消息（预览版）
- `effort` 前置参数 — 支持技能和斜杠命令
- `/fork` 已重命名为 `/branch`（保留别名）
- `SendMessage` 可自动恢复停止的代理

## MCP 服务器

### 添加服务器
```bash
--transport http      # 远程 HTTP（推荐）
--transport stdio     # 本地进程
--transport sse       # SSE
```

### 作用域
- **本地**: `~/.claude.json`（每个项目）
- **项目**: `.mcp.json`（共享/版本控制）
- **用户**: `~/.claude.json`（全局）

### 管理命令
```bash
claude mcp list    # 列出 MCP 服务器
claude mcp serve   # 将 CC 作为 MCP 服务器
```

- `/resume` — 恢复/切换会话

### 配置命令
- `/config` — 打开设置
- `/model [model]` — 切换模型（←→ effort）

- `/vim` — 开关 Vim 模式
- `/theme` — 更改主题色
- `/permissions` — 查看/更新权限
- `/effort [level]` — 设置努力程度（低/中/高）✨

- `/init` — 创建 CLAUDE.md

- `/mcp` — 管理 MCP 服务器

- `/chrome` — Chrome 集成
- `/reload-plugins` — 热重载插件

### 特殊命令
- `/btw <question>` — 附带问题（无上下文）
- `/plan [desc]` — 计划模式（+自动启动）
- `/loop [interval]` — 调度重复任务
- `/voice` — 按键说话模式（20种语言）
- `/doctor` — 诊断安装问题
- `/rc` — 启用远程控制
- `/pr-comments [PR]` — 获取 GitHub PR 评论
- `/stats` — 使用统计
- `/insights` — 分析会话报告

## 工作流与技巧

### 计划模式
- Shift+Tab: 正常 → 自动 → 计划

### 思考与努力
- Alt+T: 开关思考模式
- "ultrathink": 单轮最大努力
- Ctrl+O: 查看思考过程（详细输出）

### Git Worktree
- `--worktree name`: 每个功能分支独立隔离

### 语音模式
- `/voice`: 启用按键说话
- 空格（按住）: 录音，松开发送
- 支持 20 种语言：英语、西班牙语、法语、德语、捷克语、波兰语……

### 上下文管理
- `/context`: 使用+优化提示
- `/compact [focus]`: 压缩上下文

## 技能与代理

### 内置技能
- `/simplify` — 代码审查（3 个并行代理）
- `/batch` — 大型并行更改（5-30 个 worktree）
- `/debug [desc]` — 从调试日志排查问题
- `/loop [interval]` — 调度重复任务
- `/claude-api` — 加载 API + SDK 参考

### 自定义技能位置
- `.claude/skills/<name>/` — 项目技能
- `~/.claude/skills/<name>/` — 个人技能

### 技能前置参数
```yaml
description: 自动调用触发器
allowed-tools: 跳过权限提示
model: 覆盖技能模型
effort: 覆盖努力程度 ✨
context: fork: 在子代理中运行
$ARGUMENTS: 用户输入占位符
${CLAUDE_SKILL_DIR}: 技能自己的目录
!\`cmd\`: 动态上下文注入
```

### 内置代理
- **Explore**: 快速只读（Haiku）
- **Plan**: 计划模式研究
- **General**: 完整工具，复杂任务
- **Bash**: 终端独立上下文

### 代理前置参数
```yaml
permissionMode: default/acceptEdits/dontAsk/plan
isolation: worktree: 在 git worktree 中运行
memory: user|project: 持久记忆
background: true: 后台运行
maxTurns: 限制代理轮次
```

## CLI 与标志

### 核心命令
```bash
claude                    # 交互式
claude "q"                # 带提示
claude -p "q"             # 无头
claude -c                 # 继续上次
claude -r "n"             # 按名称恢复
claude update             # 更新
```

### 关键标志
- `--model`: 设置模型
- `-w`: Git worktree
- `-n / --name`: 会话名称
- `--add-dir`: 添加目录
- `--agent`: 使用代理
- `--allowedTools`: 预批准工具
- `--output-format`: json/stream
- `--json-schema`: 结构化
- `--max-turns`: 限制代理轮次
- `--max-budget-usd`: 成本上限
- `--console`: 通过 Anthropic Console 认证
- `--verbose`: 详细输出
- `--bare`: 极简无头模式 ✨
- `--channels`: 权限中继/MCP 推送 ✨
- `--remote`: Web 会话
- `--chrome`: Chrome

### 权限模式
- `default`: 提示
- `acceptEdits`: 自动接受编辑
- `plan`: 只读
- `dontAsk`: 拒绝除非允许
- `bypassPermissions`: 跳过所有
