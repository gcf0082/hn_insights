# Lightfeed Extractor 洞察报告

- **GitHub 星标**: 143

## 项目概述

Lightfeed Extractor 是一个 TypeScript 库，专门用于使用 LLM 和 Playwright 进行 robust 的网页数据提取。用户可以使用自然语言提示来导航网页并提取结构化数据，获得完整、准确的结果，同时具有出色的 token 效率，这对生产数据管道至关重要。

## 核心功能

### 1. 浏览器自动化（隐形模式）
- 可在本地、serverless 云或远程浏览器服务器启动 Playwright 浏览器
- 内置反机器人补丁和代理配置，避免被检测

### 2. AI 浏览器导航
- 可与 @lightfeed/browser-agent 配合使用，使用自然语言命令导航页面

### 3. LLM 友好的 Markdown
- 将 HTML 转换为 LLM 友好的 Markdown
- 支持只提取主要内容并清理 URL（移除跟踪参数）

### 4. LLM 提取功能
- 使用 JSON 模式的 LLM 根据输入的 Zod schema 提取结构化数据
- 包含 token 使用限制和追踪

### 5. JSON 恢复
- 清理和恢复失败的 JSON 输出，使复杂 schema 提取更加 robust

### 6. URL 验证
- 处理相对 URL，移除无效 URL，修复 Markdown 转义的链接

## 技术栈

- **主要语言**: TypeScript
- **浏览器自动化**: Playwright
- **LLM 集成**: LangChain (支持 OpenAI, Google Gemini, Anthropic, Ollama 等)
- **数据验证**: Zod

## 使用场景

1. **电商产品提取**: 从真实电商网站提取结构化产品数据
2. **浏览器代理导航**: 需要交互（搜索、翻页、关闭弹窗）后才能提取的场景
3. **博客/文章提取**: 使用 extractMainHtml 选项聚焦主要内容
4. **价格监控**: 竞争对手价格追踪和情报收集

## 安装和使用

```bash
npm install @lightfeed/extractor
npm install @langchain/openai  # 或其他 LLM 提供商
```

主要 API:
- `extract()`: 从内容中提取结构化数据
- `Browser`: 浏览器自动化类
- `convertHtmlToMarkdown()`: HTML 转 Markdown 工具
- `safeSanitizedParser()`: JSON 恢复工具

## 总结

Lightfeed Extractor 是一个功能强大的网页数据提取库，结合了 LLM 和浏览器自动化技术，特别适合需要从复杂网页中提取结构化数据的场景。其 robust 的错误处理、URL 验证和 JSON 恢复机制使其非常适合生产环境使用。
