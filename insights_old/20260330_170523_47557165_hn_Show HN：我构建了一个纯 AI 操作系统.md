## 项目概述

Pneuma 是一个桌面计算环境，其核心理念是"软件不需要在你需要它之前就存在"。用户开机后看到一个空白屏幕和提示符，只需描述你想要什么（如 CPU 监控器、游戏、笔记应用、数据可视化工具），程序就会在几秒钟内生成并运行。

## 技术架构

- **底层技术**: 用户输入发送给 LLM，生成自包含的 Rust 模块
- **编译过程**: Rust 模块在不到一秒内编译为 WebAssembly，然后在沙盒化的 Wasmtime 实例中进行 JIT 编译并执行
- **渲染**: 全部通过 wgpu（Vulkan/Metal/DX12）进行 GPU 渲染
- **容错机制**: 如果编译失败，错误会自动反馈给 LLM 进行修正，首次尝试成功率约 90%
- **架构**: 微内核设计，代理（Agent）在隔离的 WASM 沙盒中运行，带有用于绘制、输入、存储和网络通信的类型化 ABI

## 核心特性

1. **即时创建**: 描述你想要的应用，程序立即生成并运行
2. **代理持久化**: 生成的代理会持久化，可以重复使用
3. **IPC 通信**: 代理之间可以通过进程间通信进行交互
4. **社区分享**: 可以通过社区代理商店分享和下载代理
5. **安全隔离**: 代理崩溃不会导致系统崩溃
6. **跨平台**: 目前支持 Linux、macOS 和 Windows 的桌面应用

## 未来计划
