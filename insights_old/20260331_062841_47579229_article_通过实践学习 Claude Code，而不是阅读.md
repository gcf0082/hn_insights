## 网站概述

**Learn Claude Code Interactively** 是一个免费的交互式学习平台，帮助用户通过实践而非阅读来掌握 Claude Code。该平台提供了 11 个互动模块、终端模拟器、配置构建器和测验，无需任何安装配置即可开始学习。

---

## 主要功能与内容

### 1. 学习模块（Learning Modules）

平台提供 11 个从入门到高级的学习模块：

| 模块 | 时长 | 难度 |
|------|------|------|
| 1. Slash Commands | 30 分钟 | 入门 |
| 2. Memory & CLAUDE.md | 45 分钟 | 入门 |
| 3. Project Setup | 45 分钟 | 入门 |
| 4. Commands Deep Dive | 30 分钟 | 入门 |
| 5. Skills | 1 小时 | 中级 |
| 6-11. 更多模块 | - | 进阶 |

### 2. 核心特色

- **无需安装即可练习**：在浏览器中的终端模拟器里练习斜杠命令、钩子和技能
- **构建真实配置**：通过交互式表单生成 CLAUDE.md、钩子和插件配置
- **验证理解**：每个模块结束时有测验，答错会提供详细解释

### 3. 探索工具

- **Playground**：专用的全尺寸终端沙盒，用于练习命令
- **Config Builder**：生成 CLAUDE.md、技能、代理、钩子、MCP 服务器和插件
- **Cheat Sheet**：常用命令、快捷方式和文件的速查表
- **Feature Index**：按类型、级别、类别和模块搜索完整功能列表
- **Resources**：模板包、外部指南和社区链接
