## 项目概述

Claude Code Unpacked 是一个非官方的 Claude Code 源代码分析项目，展示了当用户在 Claude Code 中输入消息时实际发生的一切。该项目涵盖了代理循环、40多种工具、多代理编排以及未发布的功能，直接从源代码映射而来。

项目由 Zackautocracy 创建，基于 @Fried_rice 分享的 Claude Code 源代码。

---

## 核心内容

### 1. 代理循环（The Agent Loop）

从按键到渲染响应，逐步解析源代码的执行流程：

- **输入处理**: 用户输入消息或通过 stdin 管道输入
- **消息解析**: Ink 的 TextInput 组件处理键盘输入
- **历史记录**: 管理对话历史
- **系统提示**: 处理系统级指令
- **API 调用**: 与 API 交互
- **令牌处理**: 分词和令牌管理
- **工具判断**: 判断是否需要调用工具
- **循环执行**: 代理循环的核心逻辑
- **渲染输出**: 响应渲染
- **钩子机制**: 各种钩子处理
- **等待异步**: 异步操作处理

### 5. 隐藏功能

代码中存在但尚未正式发布的功能：

- **Buddy**: 生活在终端中的虚拟宠物，物种和稀有度派生自用户账户ID
- **Kairos**: 持久模式，包含每日日志、会话间的记忆整合和自主后台操作
- **UltraPlan**: 长时间规划会话，在 Opus 级模型上最长30分钟执行窗口
- **Coordinator Mode**: 引导代理将任务分解，在隔离的 git worktree 中生成并行工作者，收集结果
- **Bridge**: 从手机或浏览器控制 Claude Code，带权限批准的完整远程会话
- **Daemon Mode**: 使用 `--bg` 在后台运行会话，底层使用 tmux
- **UDS Inbox**: 会话通过 Unix 域套接字相互通信
- **Auto-Dream**: 会话之间，AI 回顾发生的事情并整理所学
