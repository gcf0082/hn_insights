# Claude Code 重写为 Bash 脚本

## 项目概述

这是一个将 Claude Code CLI 完全重写为纯 bash 脚本的实验性项目，通过移除所有依赖包实现了代码的极大精简。

## 核心亮点

1. **极简实现** - 将完整的 Claude Code 功能用 bash 脚本实现，展现了 shell 脚本的强大能力
2. **去除依赖** - 移除了所有 npm 包依赖
3. **引发思考** - 促使社区思考现代 AI 工具的"膨胀"问题

## 社区讨论焦点

- **功能膨胀 (Feature Creep)** - 作者认为这是导致 Claude Code 臃肿的主要原因
- **TUI 层的负担** - 终端用户界面贡献了大量代码膨胀
- **React 用于 TUI** - 有评论指出 TUI 本身竟然是用 React 编写的，这在技术上显得"荒谬"
- **websearch/webfetch 缺失** - 社区认为这些重要功能在精简版中缺失

## 延伸讨论

- 有开发者提到用 Python 重写大约需要 1000 行代码
- 有人期待看到用汇编语言或 Rust 重写的版本
- Claw-code 项目已在尝试用 Rust 重写 Claude Code

## 评价

这个项目虽然只是一个有趣的实验，但深刻揭示了现代 AI 工具的复杂性问题，引发了关于工程实践中"过度设计"的思考。
