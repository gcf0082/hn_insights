## 项目简介

Travel Hacking Toolkit 是一个 AI 驱动的旅行积分工具，帮助用户利用积分、里程和奖励航班进行旅行优化。该项目为 OpenCode 和 Claude Code 提供可直接使用的技能（Skills）和 MCP 服务器。

用户只需向 AI 提问，如“帮我找一张 60000 英里商务舱去东京的航班”，AI 就会跨 25+ 航空积分计划搜索奖励座位可用性，对比现金价格，检查用户积分余额，并给出最佳方案。

## 数据文件

项目包含多个数据文件，位于 `data/` 目录：

- **alliances.json** - 航空公司联盟成员资格和预订关系
- **hotel-chains.json** - 酒店连锁、子品牌、忠诚度计划、反向查询
- **partner-awards.json** - 哪些计划可预订哪些航空公司（联盟 + 双边）
- **points-valuations.json** - 4 个来源的积分/里程估值（最低/最高）
- **sweet-spots.json** - 高价值奖励兑换 + 预订窗口
- **transfer-partners.json** - 信用卡转账合作伙伴和比例

## 旅行积分工作流

核心问题：**“应该烧积分还是付现金？”**

1. **搜索奖励可用性** - Seats.aero 跨 25+ 计划
2. **搜索现金价格** - SerpAPI (Google Flights) 或 Skiplagged
3. **估算门户价值** - 门户费率是动态的
4. **对比** - 较低者获胜
5. **检查余额** - AwardWallet 确认足够
6. **预订** - 使用 Seats.aero 或 Duffel 的预订链接

## 快速开始

```bash
git clone https://github.com/borski/travel-hacking-toolkit.git
cd travel-hacking-toolkit
./scripts/setup.sh
```

5 个免费 MCP 服务器（Skiplagged、Kiwi、Trivago、Ferryhopper、Airbnb）无需任何 API 密钥即可直接使用。完整体验需要：

- **SEATS_AERO_API_KEY** - 奖励航班搜索（必需，Pro 约 $8/月）
- **SERPAPI_API_KEY** - 现金价格对比（可选，有免费额度）
