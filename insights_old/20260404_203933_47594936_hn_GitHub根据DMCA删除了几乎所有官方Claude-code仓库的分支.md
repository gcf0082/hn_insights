# Hacker News 洞察报告

## 基本信息

- **标题**: GitHub has DMCA'd nearly all forks of the official Claude-code repo
- **链接**: https://news.ycombinator.com/item?id=47594936
- **发布者**: cg505
- **发布时间**: 3小时前
- **得分**: 48
- **评论数**: 48

## 事件概述

GitHub 根据 Anthropic 提交的 DMCA 投诉，下架了几乎所有 Claude Code 官方仓库的 forks，总计约 8,100 个仓库被移除。

## 主要讨论观点

### 1. 双重标准争议
用户批评 Anthropic 存在双重标准：
- 他们抓取整个互联网的数据用于训练 AI 模型
- 但当自己的源代码被公开时却动用 DMCA 下架

### 2. 源代码泄露的讽刺性
- Claude Code 源代码被意外发布到公开的 S3 存储桶
- 有评论指出 Anthropic 抓取开源项目时完全无视其许可证
- 有人戏称"Anthropic 可以从互联网上获取公开数据，但当他们的代码成为公开数据时却不允许"

### 3. DMCA 执行的合理性质疑
- 部分评论认为这是 GitHub 执行过度
- 有用户指出 Anthropic 官方曾表示不是他们提交的 DMCA 投诉
- 原作者 cg505 表示他的 fork 根本没有包含泄露的源代码

### 4. 技术实现批评
- Claude Code 使用 ReactJS 构建 TUI 被认为是"糟糕的选择"
- 相比之下，OpenAI 的 Rust TUI 被认为更优
- 有评论认为这是" vibecoding"（AI 编程）带来的质量问题

### 5. 开源社区影响
- 有用户担心这会助长"我为人人，人人为我"的局面
- 认为最终会导致更多开源替代品的出现
- 批评微软通过 GitHub 控制开源基础设施

## 社区情绪

整体而言，评论区对 Anthropic 的做法持批评态度，认为其存在明显的双重标准。同时也引发了对 AI 公司版权立场、DMCA 滥用以及开源生态的深层思考。
