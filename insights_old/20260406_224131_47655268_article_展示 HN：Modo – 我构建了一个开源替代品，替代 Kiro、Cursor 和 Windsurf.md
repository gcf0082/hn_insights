## 项目概述

Modo 是一款开源的 AI IDE（集成开发环境），其核心特点是"先规划后编码"（plans before it codes）。该项目基于 Void 编辑器（VS Code 的一个分支）构建，在其基础上增加了多项 AI 驱动的开发功能。该项目旨在为开发者提供一个可定制、可扩展的 AI 辅助编程工具，类似于 Kiro、Cursor 或 Windsurf 等商业产品。

## 核心技术特性

### 1. 规范驱动的开发流程（Spec-Driven Development）

Modo 采用了独特的"提示词→需求→设计→任务→代码"工作流，而非传统的"提示词→代码"模式。每个规范（Spec）保存在 `.modo/specs/<name>/` 目录下，包含三个 Markdown 文件：
- `requirements.md` - 用户故事和验收标准
- `design.md` - 架构设计、组件和数据模型
- `tasks.md` - 实现步骤清单

开发者可以通过 `Cmd+Shift+S` 或会话选择器创建规范，选择功能或缺陷修复类型，以及需求优先或设计优先的工作流程。

### 2. 任务代码透镜（Task CodeLens）

在打开 `tasks.md` 文件时，每个待处理的任务都会显示一个可点击的"▶ 运行任务"按钮。文件顶部提供"运行所有待处理任务"按钮。运行中的任务会显示旋转图标，完成的任务会显示勾选标记。

### 3. 引导文件（Steering Files）

位于 `.modo/steering/` 目录下的 Markdown 文档，可将项目规则注入每次 AI 交互中。支持三种包含模式：
- `always` - 每次交互
- `fileMatch` - 当活动文件匹配 glob 模式时
- `manual` - 通过聊天中的 `/` 命令引用

### 4. 代理钩子（Agent Hooks）

位于 `.modo/hooks/` 目录下的 JSON 配置文件，可自动化代理生命周期中的操作。支持 10 种事件类型和 2 种操作类型，并内置循环依赖检测。

### 5. 自动驾驶/监督模式切换

状态栏中的切换按钮，可在自动驾驶模式（代理自主行动）和监督模式（代理等待批准）之间切换。

### 6. 并行聊天会话

支持多个聊天会话标签页，每个会话都有独立的线程、上下文和历史记录。可搜索的历史面板便于查找过去的对话。

### 7. 氛围模式和规范模式

在氛围模式下进行自由形式探索，在规范模式下触发完整的"需求→设计→任务"工作流程。

### 8. 子代理（Subagents）

可生成并行代理来执行独立的子任务，每个子代理都有独立的线程和工具访问权限。

### 9. 能力包（Powers）

可安装的知识包，包含文档、引导文件 MCP 配置。内置 TypeScript、React、测试、API 设计和 Docker 等能力包，可根据提示词中的关键词自动激活。

### 10. 其他功能

- 专用资源管理器窗格：显示规范、代理钩子、引导和能力包
- 斜杠命令：访问手动钩子、引导文件和内置命令
- 上下文注入：自动将引导文件和活动规范前置到每次 LLM 交互
- 自定义暗色主题：名为"Modo Dark"，带有青色强调色
- 伴侣角色：Modo 头像在流式响应时显示随机表情
- 完全品牌重塑：自定义图标、窗口标题、URL 协议（modo://）等
- 工作区初始化：创建完整的 `.modo/` 目录结构

## 技术架构

### 基于项目

- **Void**: 提供 AI 聊天、内联编辑、自动补全、多提供商 LLM 支持、工具调用、MCP 集成
- **VS Code**: 编辑器核心、扩展系统、终端、文件系统
- **Roo Code**: 开发过程中参考的一些模式

### 项目结构

```
.modo/                          # 每个项目的配置
├── steering/                   # 注入 AI 上下文的 markdown 规则
├── specs/                       # 结构化功能/缺陷修复工作流
├── hooks/                       # JSON 事件→操作自动化
└── settings/                   # MCP 和其他配置

src/vs/workbench/contrib/void/
├── common/                     # 核心服务
├── browser/                    # 浏览器端逻辑
└── extensions/                 # 自定义主题
```

## 快速开始

```bash
git clone https://github.com/mohshomis/modo.git
cd modo
npm install          # 需要 Node 20
npm run buildreact  # 编译 React UI
npm run watch       # 编译 TypeScript
./scripts/code.sh   # 启动 Modo
```

首次启动时，引导程序会帮助连接模型提供商。Gemini 免费层是最快的入门方式。
