## 产品概述

Claude Managed Agents 是 Anthropic 推出的托管式 AI Agent 解决方案，旨在帮助企业快速构建和部署生产级 AI Agent。该产品于2026年4月8日发布公开测试版，使团队能够将开发周期从数月缩短至数天，实现"10倍速"的生产力提升。

---

## 核心功能

### 1. 生产级 Agent 基础设施
- **安全沙箱执行**：内置代码隔离环境，确保安全运行
- **身份认证与工具执行**：完整的认证和授权机制
- **长时运行会话**：支持数小时自主运行，离线后状态和输出可恢复

### 2. 多 Agent 协调（研究预览）
- Agent 可自主启动和指挥其他 Agent
- 支持复杂任务的并行处理

### 3. 信任治理
- ** scoped permissions**：精细化权限控制
- **身份管理**：完整的身份验证体系
- **执行追踪**：端到端的操作审计
