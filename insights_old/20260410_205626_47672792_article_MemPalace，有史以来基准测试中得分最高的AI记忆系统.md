## 技术架构

### Palace 记忆结构

MemPalace 采用「宫殿「记忆法，灵感来自古希腊演说家：

- **Wings（翼）** — 项目或人物分类
- **Halls（厅）** — 记忆类型（决策、事件、发现、偏好、建议）
- **Rooms（房间）** — 具体主题
- **Closets（衣柜）** — 摘要指向原始内容
- **Drawers（抽屉）** — 原始verbatim内容

### 存储方式

- 使用 ChromaDB 进行语义搜索
- 原始verbatim存储，不做摘要提取
- 96.6%的得分来自这种原始模式

### AAAK 方言（实验性）

- 损耗性缩写系统，用于压缩重复实体
- 可被任何LLM直接读取（Claude, GPT, Gemini, Llama等）
- 注意：AAAK模式在LongMemEval上得分较低（84.2%）

## 使用方式

### 安装

```bash
pip install mempalace
```

### 初始化

```bash
mempalace init ~/projects/myapp
```

### 挖掘数据

```bash
mempalace mine ~/projects/myapp
mempalace mine ~/chats/ --mode convos
```

### 搜索

```bash
mempalace search "why did we switch to GraphQL"
```
