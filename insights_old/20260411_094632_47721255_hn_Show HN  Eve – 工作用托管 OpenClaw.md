## 项目概述

Eve 是一个托管式 AI Agent 工具，旨在为日常工作提供无需自托管的 OpenClaw 功能。其设计理念是将 AI 定位为"有用的同事"而非个人助理。

### 核心功能

**隔离环境**: 在独立的 Linux 沙箱中运行（2 vCPU、4GB RAM、10GB 磁盘），配备真实文件系统、无头浏览器、代码执行能力，并可连接 1000+ 种服务。

**任务执行**: 用户只需下达任务，Eve 会在后台自动完成工作。

**交互界面**: 
- Web 应用界面，实时展示工作过程（Agent 启动、文件编写、CLI 使用）
- iMessage 集成，支持异步任务执行，完成后自动通知

**底层架构**:
- 使用 Claude Opus 4.6 作为编排器
- 根据子任务类型路由到特定领域模型（浏览、编码、研究、媒体生成）
- 并行子 Agent 通过共享文件系统协调工作
- 跨会话持久化记忆，语境可累积

**预装技能**: 内置多种工作角色技能（销售、营销、金融等），可在运行时执行相应任务。

### 用户反馈

**积极反馈**:
- 有用户通过 iMessage 和 Web UI 成功下载医疗数据集并进行分析
- 赞赏其允许在 OpenClaw 功能上进行安全实验
- 免费积分对试用很有帮助

**安全担忧**:
- 连接邮箱、Slack、GitHub 等服务时缺乏足够的开发者信息
- 很少有网站信息却要求如此高的权限让人难以信任
- 链接直接下载文件的行为引发疑虑

**争议讨论**:
- 在主域托管用户内容被指出存在安全隐患
- 自主 LLM Agent 是否已准备好用于商业环境引发讨论
- ".new"域名政策合规性问题

### 技术特点

- 实时监控：用户可观看 Agent 工作过程
- 多服务集成：支持 1000+ 服务连接
- 角色扮演：可灵活扮演不同职业角色
- 持久记忆：跨会话累积上下文

**当前状态**: 新用户获得 $100 积分用于试用。

Eve 代表了 AI Agent 托管服务的新趋势，将强大的自主Agent能力包装成易于使用的服务。但围绕数据安全和信任的问题仍然是用户的主要顾虑。
