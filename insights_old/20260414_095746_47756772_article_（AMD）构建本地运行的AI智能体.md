## 概述

AMD GAIA 是一个开源框架，用于在 Python 和 C++ 中构建完全运行在本地硬件上的 AI 代理。该框架支持在 AMD Ryzen AI 设备上进行 NPU 和 GPU 加速，实现隐私优先的本地 AI 处理，无需云依赖，不依赖外部 API 密钥。

## 核心特性

- **本地推理**: 所有处理均在设备端完成
- **无云依赖**: 无需 API 密钥或外部服务
- **双语言支持**: 提供完整的 Python 和 C++ SDK
- **AMD 优化**: 支持 Ryzen AI 上的 NPU 和 GPU 加速

## 快速入门

### Agent UI（最快方式）
通过 npm 安装，两条命令即可本地运行 AI 代理。

### Python 快速开始
安装 GAIA，启动 Lemonade Server，运行第一个代理。

### C++ 快速开始
构建原生 C++17 代理二进制文件，无需 Python 运行时。

## 主要功能

### Python 功能
| 功能 | 描述 |
|------|------|
| Agent UI | 隐私优先的桌面聊天，支持拖放文档问答 |
| 文档问答（RAG） | 检索、索引和回答本地 PDF、代码和文本文件问题 |
| 语音转语音 | 离线语音交互管道（Whisper ASR + Kokoro TTS） |
| 代码生成 | 多文件代码生成、验证、测试和编排 |
| 图像生成 | 多模态图像生成，LLM 提示增强 |
| MCP 集成 | 通过模型上下文协议连接外部工具 |
| 代理路由 | 跨多个专业代理的智能请求路由 |

### C++ 功能
- **系统健康代理**: MCP 驱动的诊断（CPU、内存、磁盘、网络、GPU 监控）
- **Wi-Fi 故障排除**: 诊断和解决无线连接问题
- **自定义代理**: 使用工具注册和状态管理构建自定义 C++17 代理
- **MCP 集成**: 连接 MCP 服务器以扩展工具访问
