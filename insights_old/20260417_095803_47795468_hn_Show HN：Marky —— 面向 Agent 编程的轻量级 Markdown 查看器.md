## 洞察摘要

在 AI 编程代理时代，作者发现自己在审查 Markdown 文件上花费了大量时间。无论是让代理生成的计划还是文档，似乎阅读 Markdown 的时间比写代码的时间还多。作者尝试过多种解决方案，如 Obsidian，但发现其 Vault 系统限制性较强，TUI 解决方案也不够友好阅读。因此开发了 **Marky**。

**Marky** 是一款轻量级桌面应用，可轻松阅读和跟踪 Markdown 文件。它还提供便捷的 CLI，只需运行 `marky FILENAME` 即可打开指定的 md 文件。作者过去一周每天都在使用，非常喜欢，所以分享给大家。

## 核心功能

- 轻量级 Markdown 阅读器
- CLI 快速打开文件
- 每日使用体验良好
