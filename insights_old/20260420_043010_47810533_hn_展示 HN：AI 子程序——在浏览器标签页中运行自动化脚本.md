## 项目概述

AI Subroutines 是 rtrvr.ai 推出的创新工具，允许用户记录浏览器任务并保存为可调用的工具，实现零 token 成本、零 LLM 推理延迟和零错误的自动化执行。该项目将传统浏览器代理方式进行了根本性改进——不再对每个操作进行推理，而是让用户录制一次操作流程，随后可无限次调用。

核心工作流程包括：通过浏览器扩展记录用户操作，利用 DOM 事件时序和方法将约 300 个请求精简为约 5 个脚本，并使用 volatile GraphQL 请求处理动态内容。用户只需指向一个包含 500 行的电子表格进行一次 LLM 调用，即可分配参数并触发 500 个子程序并行执行。

该工具的主要应用场景包括：录制 Instagram DM 发送流程后实现零 token 成本自动发送消息；创建获取站点目录最新产品的脚本，通过直接 GraphQL 调用获取数千商品信息；设置表单填写程序，AI 可根据当前页面上下文自动推断参数。

团队认为浏览器代理未能普及的根本原因是：对于重复性任务来说，每次都走推理流程是不必要的。更好的方案是一次录制，然后让 LLM 生成可利用所有浏览器 API 的脚本。

技术层面，团队发现 Gemini Flash 是代理操作和代码编写的最佳平衡点，即使是 Flash-Lite 也表现不佳。他们正在研究自愈机制，如回退到实时网页代理并重写脚本。

为建立信任，项目方选择开源客户端代码，将 rtrvr 的 DOM 智能集成到用户自己的网页中。可在 GitHub 查看源码：https://github.com/rtrvr-ai/rover

 monetization 采用简单的订阅制收费模式。如用户倾向云端执行而非本地设备，也可选择云沙盒运行。
