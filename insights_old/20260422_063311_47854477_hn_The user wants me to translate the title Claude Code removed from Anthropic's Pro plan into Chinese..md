## 事件概述

Anthropic 已将 Claude Code 从 $20/月的 Pro 订阅计划中移除。用户需要至少升级到 Max 计划（$100/月）才能继续使用 Claude Code。此举引发了 Hacker News 社区的广泛讨论和负面反馈。

---

## 主要观点

### 1. 开发者情绪剧烈波动

一位用户（robertkarl）分享了他个人的"过山车"经历：

- **1月**：从 $20 计划开始，体验到 AI 编程的价值后升级到 $200 计划
- **4月**：因 Claude 4.6 的幻觉和懒惰问题，以及 Anthropic 官方回应态度令人不悦，降级到 $100 计划
- **最近一周**：因 Codex 5.4 的 $20 计划表现出色，再次降级回 $20 计划

此外，他提到 Anthropic 的源代码泄露事件以及与当前政府机构的互动也影响了其声誉。该用户表示自己从"最坚定的支持者"变成了"不再支持"。

### 2. 定价信息混乱

社区对 Anthropic 的定价策略表示困惑：

- **Claude Code 产品页面**仍然显示 Pro 计划包含 Claude Code
- **定价页面**显示 Pro 计划不包含 Claude Code
- **支持文章**已将 Claude Code 从 Pro 计划功能中移除
- **Anthropic 的官方 "Fin" 机器人**表示 Claude Code 需要 Team 或 Enterprise 计划

有用户（eleventen）指出："他们在营销网站上存在明显矛盾...他们正在所有页面更改，但还没有更新那一页。"

### 3. 替代方案讨论

大量用户表示将转向竞争产品：

| 替代方案 | 评论 |
|---------|------|
| **OpenAI Codex** | 包含在 $20 计划中，代码能力强 |
| **Cursor** | $20 计划可使用多种模型，Composer 2 表现优秀 |
| **Kimi/GLM** | 中国模型的低费用编程计划 |
| **本地模型 (Qwen)** | 在 MacBook 上本地运行，成本低 |
| **MiniMax** | $10/月，约 3 倍于 Claude Pro 的使用量 |

用户（operatingthetan）预测："我们不可避免地会转向使用廉价的中国模型如 Kimi、GLM 和 MiniMax。在 3-6 个月内它们将达到 Opus 4.6 的水平。"

### 4. 经济考量

多位用户讨论了 LLM 公司的盈利能力问题：

- 用户（Yizahi）指出："LLM 怪物深陷亏损...只有两个披露财务的中国公司证实了这一点"
- 用户（chewz）认为："这对独立开发者和爱好者来说已经结束了"
- 用户（jfrbfbreudh）每月支付 $200 计划但实际使用价值 $3000 的 API 调用

### 5. 企业用户 vs 个人用户

Anthropic 的策略被认为偏向企业客户：

- 用户（mrhottakes）直言："他们不在乎能写大支票的人"
- 用户（fg137）认为个人用户"几乎无关紧要"
- 但用户（tayo42）反驳："个人用户才是推动企业采用新工具的人"

### 6. 对 Anthropic 的批评

- **产品管理混乱**：有用户嘲笑 Anthropic 和微软一样在定价决策上令人困惑
- **沟通能力差**：Anthropic 的消息传递被认为糟糕
- **失去开发者信任**：用户从热情支持者变为取消订阅
