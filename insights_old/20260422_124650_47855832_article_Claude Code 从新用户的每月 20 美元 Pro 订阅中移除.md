# 洞察报告：Anthropic 从 Pro 订阅计划中移除 Claude Code

**洞察链接**: https://www.wheresyoured.at/news-anthropic-removes-pro-cc/

**来源**: Ed Zitron's Where's Your Ed At

**发布日期**: 2026年4月21日

**作者**: Ed Zitron

---

## 摘要

2026年4月21日下午，Anthropic 从其每月20美元的 "Pro" 订阅计划中移除了 AI 编码工具 Claude Code 的访问权限。这一变化引发了广泛关注，随后 Anthropic 又迅速恢复了原有的访问权限。

## 事件经过

### 初始变化
- Anthropic 在多个定价页面上移除了 Pro 用户对 Claude Code 的访问权限
- 当时的帮助文档显示访问 Claude Code 需要 "Max 计划"，而非之前的 "Pro 或 Max 计划"
- 现有的 Pro 用户通过 Claude 网页应用似乎仍可访问

### 官方回应
Anthropic 增长负责人 Amol Avasare 在 Twitter 上声称这是 "针对约2%新专业用户的小规模测试"。当被问及为何修改帮助文档时，他未作回应。

### 后续发展
- 在公众关注后，Anthropic 迅速恢复了网站和帮助文档的原状
- 然而，Amol Avasare 暗示未来可能还会有更多变化

## 背景分析

Anthropic 近期已采取多项成本控制措施：

1. 不再提供固定的速率限制，改为按百万令牌计费
2. Claude Max 计划在 Claude Code 和 Claude Cowork 之前发布，主要设计为重度聊天使用场景

这些迹象表明，订阅层级的进一步变化可能即将到来。

## 影响

这一事件引发了对以下问题的关注：
- AI 订阅服务的定价策略是否可持续
- 用户权益是否能够得到保障
- 未来是否会出现更多限制

## 结论

虽然 Anthropic 迅速恢复了原有访问权限，但这一事件反映出 AI 公司在运营成本方面面临的压力。随着 AI 使用场景的扩展，订阅服务的定价和功能调整可能成为常态。
