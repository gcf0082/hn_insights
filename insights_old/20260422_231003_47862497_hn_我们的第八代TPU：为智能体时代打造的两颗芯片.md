## 核心摘要

Google在Cloud Next 2026大会上发布了第八代Tensor Processing Unit（TPU），包括两款专用芯片：TPU 8t（训练芯片）和TPU 8i（推理芯片）。这两款芯片专门为智能体（Agent）时代设计，旨在应对AI智能体连续推理、规划、执行和学习的复杂工作负载。TPU 8t专注于大规模模型训练，TPU 8i专注于低延迟推理，两者均可提供上一代产品2倍的每瓦性能。

---

## 技术规格

### TPU 8t（训练芯片）

| 特性 | 规格 |
|------|------|
| 单Superpod规模 | 9,600个芯片 |
| 共享高带宽内存 | 2 PB |
| 芯片间带宽 | 上一代2倍 |
| 计算性能 | 121 ExaFlops |
| 存储访问速度 | 10x提升 |
| 网络架构 | Virgo Network |

TPU 8t旨在将前沿模型开发周期从数月缩短至数周，通过最大化计算吞吐量、共享内存和芯片间带宽，同时保持最佳能效。

### TPU 8i（推理芯片）

| 特性 | 规格 |
|------|------|
| 高带宽内存 | 288 GB |
| 片上SRAM | 384 MB（上一代3倍） |
| 主机CPU | 自研Axion ARM |
| ICI带宽 | 19.2 Tb/s |
| 片上延迟 | 降低最高5倍 |
| 性价比 | 提升80% |

TPU 8i专门针对低延迟推理进行了优化，配备更大内存带宽，能够处理需要快速响应的推理工作负载。

## 社区讨论要点

### 1. Google与NVIDIA的竞争格局

> "At this point, when you are doing big AI you basically have to buy it from NVidia or rent it from Google."

多位评论者认为，在大型AI领域，NVIDIA和Google是两大主导力量。Google的优势在于从芯片到系统的全栈垂直整合，能够在整个数据中心层面进行优化，这在成本效率上具有显著优势。

### 2. Gemini模型的实际表现

讨论中关于Gemini模型的意见分歧较大：

- **正面**: Gemini在日常任务中表现良好，响应速度快，token使用量显著少于ChatGPT和Claude
- **负面**: Gemini CLI体验不佳，代码任务表现不如Claude Code和Codex；推理和执行能力有待提升

> "I use Claude Code all day and use Gemini CLI for personal projects and I don't see the huge gap that other people seem to talk about."

### 3. Google的长期优势

评论者指出Google的多重优势：

- 拥有从芯片到应用的完整技术栈
- 超过10亿用户的使用场景
- 多年AI研究积累（包括Transformer架构的原创发明）
- 垂直整合带来的成本控制能力

> "Google has been going from strength to strength quietly in the background capturing consumer market share"

### 4. 对AI泡沫的不同看法

关于AI行业是否存在泡沫，观点不一：

- **认为有泡沫**: 消费者硬件不断进步，小型模型将满足大部分需求
- **认为无泡沫**: 需求持续超过供应，所有厂商都在面临扩展问题

> "Everyone is having scaling issues with more demand than they can cover."

### 5. 硬件专业化趋势

评论者注意到TPU在训练和推理之间进行专业化分离的趋势：

- 训练任务受计算能力限制
- 推理任务受内存带宽限制
- Cerebras和Groq等厂商也在开发专用推理加速器

## 关键技术亮点

### 1. 智能体时代的基础设施

在智能体时代，AI模型需要：
- 对问题进行推理
- 执行多步骤工作流
- 从自身行为中持续学习

���给基础设施带来了新的需求，需要适应不断演进的模型架构。

### 2. Boardfly拓扑结构

TPU 8i采用创新的Boardfly架构，将最大网络直径减少50%以上，确保系统作为统一低延迟单元工作。

### 3. 能效提升

TPU 8t和TPU 8i提供高达2倍的每瓦性能提升。同时，数据中心的计算能力比五年前提升了6倍。

### 4. 液冷技术

第四代液冷技术支持空气冷却无法达到的性能密度，这对于大规模AI训练至关重要。

## 总结

Google第八代TPU代表了其十年芯片开发的结晶，通过专业化设计和垂直整合，在智能体时代为AI工作负载提供了强大的基础设施支撑。尽管在软件层面（如Gemini模型）仍面临竞争挑战，但Google在硬件层面的持续创新使其在AI基础设施竞争中保持有利地位。
