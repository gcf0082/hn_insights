# MeshCore 分裂事件洞察

**洞察链接**: https://blog.meshcore.io/2026/04/23/the-split
**发布日期**: 2026年4月23日
**来源**: MeshCore Blog

## 事件概述

MeshCore 开发团队自项目启动以来一直致力于构建 MeshCore 生态系统。截至目前，已发布超过85个版本的 MeshCore Companion、Repeater 和 Room Server 固件，支持超过75种硬件变体，所有这些均由人工手动 crafted 开发。

## 核心冲突

团队成员 Andy Kirby 决定广泛使用 Claude Code，并试图接管 MeshCore 生态系统的所有组件：独立设备、移动应用、网页刷机工具和网页配置工具。关键问题是，Andy 将这一重要细节保密——这些产品绝大多数都是"vibe coded"（ vibe 编程）。

团队在 Discord 进行了一项关于 AI 和信任的民意调查，结果显示社区对 AI 生成固件的信任度存在显著分歧。

更严重的是，团队发现 Andy 在3月29日以个人名义申请了 MeshCore 商标，而未告知团队其他成员。经过多次讨论意图未果，目前团队与 Andy 之间已中断沟通。

## 品牌争议

"官方" MeshCore 品牌使用权目前存在争议。 Andy 坚信他拥有品牌，并在其 MeshOS 产品线中大量使用"官方"字样。

实际上，唯一"官方"的 MeshCore 是 GitHub 仓库——这是判断什么是 MeshCore 的真相来源。而 Andy 从未对该仓库做出过贡献。

自内部分裂后，团队推出了 meshcore.io 网站，因为 Andy 控制着 meshcore.co.uk 网站和原始 Discord 服务器。

## 项目发展现状

MeshCore 项目自2025年1月启动以来发展迅速：
- 官方 MeshCore Map 显示全球已有38,000+节点
- 官方 MeshCore 应用拥有超过100,000+活跃用户（ Android 和 iOS ）

## 未来方向

核心团队将继续推进 meshcore.io 网站、固件功能开发、bug 修复、PR 管理和开发者讨论等工作。

新网站将发布变更日志、博客文章和技术文档，涵盖：
- Scott：项目创始人、主固件工程师、Ripple 固件开发者
- Recrof：官方 MeshCore Map 开发者、固件刷机工具专家
- Liam Cottle：官方 MeshCore App 开发者
- FDLamotte：Python 工具和 STM32 固件变体维护者
- Oltaco：OTA Fix bootloader 开发者

## 结论

团队成员 Scott、Liam、Recrof、FDLamotte 和 Oltaco 仍致力于设计和开发高质量的人工编写软件。团队呼吁用户更新书签，访问新网站获取官方信息、版本发布和技术文档。
