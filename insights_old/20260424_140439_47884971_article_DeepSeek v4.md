## 一、API 概述

DeepSeek API 采用与 OpenAI/Anthropic 兼容的 API 格式。通过修改配置，可以使用 OpenAI/Anthropic SDK 或兼容 OpenAI/Anthropic API 的软件来访问 DeepSeek API。

### 1.1 接入参数

| 参数 | 值 |
|------|-----|
| base_url (OpenAI) | `https://api.deepseek.com` |
| base_url (Anthropic) | `https://api.deepseek.com/anthropic` |
| api_key | 需要在 [DeepSeek 平台](https://platform.deepseek.com/api_keys) 申请 |

### 1.2 可用模型

| 模型名称 | 说明 |
|----------|------|
| `deepseek-v4-flash` | 最新版本，快速响应 |
| `deepseek-v4-pro` | 最新版本，高性能 |
| `deepseek-chat` | 将在 2026/07/24 停用 |
| `deepseek-reasoner` | 将在 2026/07/24 停用 |

**注意**: `deepseek-chat` 和 `deepseek-reasoner` 将于 2026/07/24 停用。为保持兼容性，它们分别对应 `deepseek-v4-flash` 的非思考模式 和思考模式。

## 二、核心功能

### 2.1 思考模式 (Thinking Mode)

DeepSeek API 支持思考模式，可以通过以下参数启用：

```json
{
  "thinking": {"type": "enabled"},
  "reasoning_effort": "high"
}
```

- `thinking`: 启用思考模式
- `reasoning_effort`: 设置推理强度（支持 high/medium/low）

### 2.2 流式响应

支持将 `stream` 参数设置为 `true` 以获取流式响应。

### 2.3 其他功能

- **多轮对话**: 支持多轮对话功能
- **Chat Prefix Completion** (Beta): 前缀补全功能
- **FIM Completion** (Beta): 填充中间completion功能
- **JSON Output**: 支持 JSON 模式输出
- **Tool Calls**: 支持工具调用
- **Context Caching**: 支持上下文缓存 (KV Cache)
- **Coding Agents**: 支持与编程代理集成
- **Anthropic API**: 兼容 Anthropic API 格式

## 三、调用示例

### 3.1 cURL

```bash
curl https://api.deepseek.com/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer ${DEEPSEEK_API_KEY}" \
  -d '{
        "model": "deepseek-v4-pro",
        "messages": [
          {"role": "system", "content": "You are a helpful assistant."},
          {"role": "user", "content": "Hello!"}
        ],
        "thinking": {"type": "enabled"},
        "reasoning_effort": "high",
        "stream": false
      }'
```

### 3.2 Python

```python
from openai import OpenAI

client = OpenAI(
    api_key=os.environ.get('DEEPSEEK_API_KEY'),
    base_url="https://api.deepseek.com"
)

response = client.chat.completions.create(
    model="deepseek-v4-pro",
    messages=[
        {"role": "system", "content": "You are a helpful assistant"},
        {"role": "user", "content": "Hello"},
    ],
    stream=False,
    reasoning_effort="high",
    extra_body={"thinking": {"type": "enabled"}}
)
print(response.choices[0].message.content)
```

### 3.3 Node.js

```javascript
import OpenAI from "openai";

const openai = new OpenAI({
    baseURL: 'https://api.deepseek.com',
    apiKey: process.env.DEEPSEEK_API_KEY,
});

const completion = await openai.chat.completions.create({
    messages: [{ role: "system", content: "You are a helpful assistant." }],
    model: "deepseek-v4-pro",
    thinking: {"type": "enabled"},
    reasoning_effort": "high",
    stream: false,
});
console.log(completion.choices[0].message.content);
```

## 五、总结

DeepSeek API 是一个功能强大的 AI 服务平台，其主要特点包括：

1. **兼容性高**: 与 OpenAI/Anthropic API 格式兼容，便于现有应用迁移
2. **模型丰富**: 提供 V4 系列模型，支持思考模式和推理
3. **功能全面**: 支持多轮对话、工具调用、JSON 输出、上下文缓存等高级功能
4. **多语言支持**: 提供中文和英文文档
5. **即将停用**: 老模型 (deepseek-chat, deepseek-reasoner) 将于 2026/07/24 停用，建议迁移到 V4 系列

对于开发者而言，DeepSeek API 提供了灵活的接入方式和丰富的功能，适合构建各类 AI 应用。
