# 洞察报告：使用 Claude Code 自动化管理财务

## 基本信息

- **来源链接**: https://driggsby.com/blog/claude-code-routine-watch-my-finances
- **发布日期**: 2026年4月23日
- **作者**: Matt May
- **标签**: Claude Code, 自动化, 财务管理, MCP

## 文章摘要

本文介绍了作者如何使用 Claude Code 的 Routines 功能搭建每日财务自动化管理流程。作者首先描述了他之前尝试使用 cron-job 配合 Codex CLI 自动登录银行、信用卡和投资账户提取余额和交易记录的经历。虽然这个方案最初效果不错，但经常会因为浏览器渲染问题、意外的双因素认证或 GPT 生成的邮件格式变化而导致失败。最终，作者开发了 Driggsby——一个基于 MCP 的服务器，通过 Plaid 连接金融机构，提供了获取余额、交易、投资、贷款等信息的工具。

作者重点介绍了如何利用 Claude Code Routines 实现每日财务邮件自动化。通过简单的提示词配置，他设置了一个每天早自动运行的Routine，自动从 Driggsby 获取财务数据并发送格式化邮件到邮箱。他还为妻子创建了自定义的每日邮件，让两人可以分别关注自己感兴趣的财务信息。

在此基础上，作者进一步扩展了自动化场景：每周运行交易异常检测 Routine 来识别信用卡欺诈或订阅价格变化；监控大额支出异常；分析投资组合和订阅服务等。整个过程无需编写复杂的代理循环代码或部署基础设施，只需编写提示词即可。

## 核心要点

1. **Routines 的便利性**：Claude Code Routines 让用户可以轻松创建自动化代理，无需处理基础设施部署
2. **MCP 连接器**：Driggsby 作为 MCP 服务器，提供了访问金融机构数据的标准化接口
3. **Gmail 集成**：通过 Gmail connector 创建邮件草稿，配合自定义 email_me() 工具发送
4. **可扩展性**：可以快速创建新的 Routine 来满足不同的财务监控需求
5. **非技术用户友好**：作者的妻子（一名CPA）也能轻松创建自己的自动化流程

## 应用场景

- 每日账户余额和净资产汇总邮件
- 信用卡交易异常检测（识别重复扣款、价格变化等）
- 大额支出监控和预警
- 投资组合变化追踪
- 订阅服务分析和监控

## 总结

Claude Code Routines 的出现使得创建日常自动化任务变得极其简单。通过 MCP 连接器将数据工具接入，用户只需编写提示词即可实现复杂的自动化流程。这种低门槛的自动化方式值得更多用户尝试，特别是对于需要定期关注财务状况但又不希望手动整理数据的个人或家庭。
