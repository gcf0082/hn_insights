## 项目概述

YourMemory是一个为AI代理提供持久记忆层的开源项目，基于人类记忆科学构建。它的核心创新在于实现了艾宾浩斯遗忘曲线（Ebbinghaus forgetting curve）机制，使记忆能够像人类记忆一样——重要的内容记住，遗忘的内容自动淡出，过时的事实自动更新。

## 核心特性

### 1. 艾宾浩斯遗忘曲线

Memory strength decays exponentially — but importance and recall frequency slow that decay. 该项目实现了基于科学遗忘曲线的记忆衰减机制：

```
effective_λ = base_λ × (1 - importance × 0.8)
strength    = importance × e^(−effective_λ × days) × (1 + recall_count × 0.2)
score       = cosine_similarity × strength
```

- 频繁被回忆的记忆抵抗衰减
- 强度低于0.05的记忆每24小时自动清除

### 2. 混合检索：向量 + 图

检索分为两轮进行：

**第一轮——向量搜索**：对所有记忆进行余弦相似度计算，返回相似度高于阈值的top-k结果。

**第二轮——图扩展**：从第一轮的种子节点进行BFS遍历，通过语义边（余弦相似度≥0.4）发现共享上下文但词汇不同的记忆。

### 3. 记忆分类控制衰减速度

不同类别的记忆有不同的存活时间：

| 类别 | 无recall存活时间 | 用途 |
|------|------------------|------|
| strategy | ~38天 | 成功的模式 |
| fact | ~24天 | 偏好、身份 |
| assumption | ~19天 | 推断的上下文 |
| failure | ~11天 | 错误、环境特定问题 |

## 性能对比

在LoCoMo-10基准测试（1,534个问答对，10个多会话对话）上的表现：

| 系统 | Recall@5 | 95% CI |
|------|----------|--------|
| **YourMemory** (BM25 + vector + graph + decay) | **59%** | 56–61% |

## 技术栈

| 组件 | 角色 |
|------|------|
| DuckDB | 默认向量数据库——零设置，原生余弦相似度 |
| NetworkX | 默认图后端 |
| sentence-transformers | 本地嵌入（all-mpnet-base-v2, 768维） |
| spaCy | 本地NLP用于去重和SVO三元组提取 |
| APScheduler | 自动24小时衰减任务 |
| PostgreSQL + pgvector | 可选——用于团队或大型数据集 |
| Neo4j | 可选图后端 |

## 多代理记忆

多个代理可以共享同一个YourMemory实例——每个代理都有隔离的私有记忆和对共享上下文的有控制访问。

```python
from src.services.api_keys import register_agent

result = register_agent(
    agent_id="coding-agent",
    user_id="sachit",
    can_read=["shared", "private"],
    can_write=["shared", "private"],
)
# → result["api_key"] — ym_xxxx, shown once only
```

## 快速开始

### 安装

```bash
pip install yourmemory
```

### 设置（仅一次）

```bash
yourmemory-setup
```

下载spaCy语言模型并在`~/.yourmemory/memories.duckdb`初始化本地数据库。

### 集成客户端

支持多种AI客户端：
- Claude Code
- Claude Desktop
- Cline (VS Code)
- Cursor
- OpenCode
- Windsurf, Continue, Zed等任何MCP兼容客户端

## MCP工具

| 工具 | 调用时机 | 功能 |
|------|----------|------|
| recall_memory(query) | 每个任务开始 | 检索按相似度×强度排序的相关记忆 |
| store_memory(content, importance) | 学到新东西后 | 嵌入并存储，附带生物衰减 |
| update_memory(id, new_content) | 记忆过时 | 重新嵌入并替换 |

## 总结

YourMemory是一个创新性的AI代理记忆系统，通过引入艾宾浩斯遗忘曲线机制，使AI记忆能够像人类记忆一样自然地衰减和更新。其混合检索（向量+图）架构在LoCoMo基准测试中展现了优异的性能，比Zep Cloud高出2倍的Recall@5。该项目安装简单，支持多种主流AI客户端，是AI代理实现持久记忆的优秀选择。
