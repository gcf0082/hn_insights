## 项目概述

Dirac 是一个专注于高效和上下文精简的 AI 编程代理。该项目旨在在保持代码质量的同时大幅降低 API 成本。与其他代理相比，Dirac 可节省 **50-80%** 的 API 成本，同时提高代码质量。

## 核心特性

### 1. 哈希锚定编辑（Hash-Anchored Edits）
Dirac 使用稳定的行哈希来精确定位编辑位置，避免传统行号编辑带来的"翻译丢失"问题。这使得编辑更加精确，减少了误编辑的风险。

### 2. AST 原生精度
内置对语言语法（TypeScript、Python、C++ 等）的深度理解，允许 Dirac 执行结构化操作，如函数提取或类重构，达到 100% 准确率。

### 3. 多文件批处理
Dirac 可以在单次 LLM 调用中处理和编辑多个文件，显著降低延迟和 API 成本。

### 4. 高带宽上下文
优化的上下文管理确保 LLM 始终拥有最相关信息，避免浪费令牌。

### 5. 自主工具使用
可以读取/写入文件、执行终端命令、使用无头浏览器等，同时保持用户对工作流程的控制。

## 安装和使用

### VS Code 扩展
从 VS Code Marketplace 安装 Dirac。

### CLI（终端）
```bash
npm install -g dirac-cli
dirac auth
dirac "分析这个项目的架构"
```

### 常用命令
- `dirac "prompt"`: 启动交互式任务
- `dirac -p "prompt"`: 计划模式
- `dirac -y "prompt"`: 自动批准模式

## 技术亮点

1. **无 MCP 依赖**: 不使用 Model Context Protocol，简化架构
2. **上下文长度优化**: 通过保持精简的上下文长度来提高准确性和成本效率
3. **大规模并行操作**: 支持同时编辑多个文件
4. **多种 API 提供商支持**: Anthropic、OpenAI、OpenRouter、Gemini、Groq、Mistral 等
